## Variablen 1

In dieser Aufgabe möchten wir, dass Sie:

- Eine Variable namens `myName` deklarieren.
- `myName` mit einem geeigneten Wert initialisieren, auf einer separaten Zeile (Sie können Ihren tatsächlichen Namen verwenden oder etwas anderes).
- Eine Variable namens `myAge` deklarieren und sie auf derselben Zeile initialisieren.
