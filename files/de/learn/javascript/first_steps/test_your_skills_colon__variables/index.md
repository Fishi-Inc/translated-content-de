## Variablen 1

In dieser Aufgabe möchten wir, dass Sie:

- Deklarieren Sie eine Variable namens `myName`.
- Initialisieren Sie `myName` mit einem geeigneten Wert auf einer separaten Zeile (Sie können Ihren tatsächlichen Namen oder etwas anderes verwenden).
- Deklarieren Sie eine Variable namens `myAge` und initialisieren Sie sie auf derselben Zeile mit einem Wert.
