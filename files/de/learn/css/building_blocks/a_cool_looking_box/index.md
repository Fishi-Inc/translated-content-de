## Projektbeschreibung

Ihre Aufgabe ist es, eine coole, schicke Box zu erstellen und den Spaß zu erkunden, den wir mit CSS haben können.

### Allgemeine Aufgaben

- Wenden Sie das CSS auf das HTML an.

### Die Box gestalten

Wir möchten, dass Sie die bereitgestellte {{htmlelement("div")}} gestalten und ihr Folgendes geben:

- Eine angemessene Breite für eine große Box, sagen wir etwa 200 Pixel.
- Eine angemessene Höhe für eine große Box, wobei der Text vertikal zentriert wird.
- Die Box horizontal zentrieren.
- Den Text innerhalb der Box zentrieren.
- Eine leichte Erhöhung der Schriftgröße auf etwa 17-18 Pixel berechnet. Verwenden Sie rems. Schreiben Sie einen Kommentar, wie Sie den Wert ermittelt haben.
- Eine Grundfarbe für das Design. Geben Sie der Box diese Farbe als Hintergrundfarbe.
- Eine kontrastierende Farbe für den Text und einen schwarzen Textschatten.
- Einen ziemlich subtilen Eckradius.
- Einen 1-Pixel starken soliden Rahmen mit einer Farbe, die der Grundfarbe ähnelt, jedoch in einem etwas dunkleren Farbton.
- Einen linearen halbtransparenten schwarzen Verlauf, der zur unteren rechten Ecke verläuft. Lassen Sie ihn am Anfang völlig transparent sein, und erhöhen Sie die Deckkraft bis zu etwa 0,2 bei 30%, wobei die Farbe bis zum Ende gleich bleibt.
- Mehrere Box-Schatten. Geben Sie ihm einen Standard-Boxschatten, um die Box leicht von der Seite abgehoben aussehen zu lassen. Die anderen beiden sollten eingesetzte Boxschatten sein – ein halbtransparenter weißer Schatten nahe der oberen linken Ecke und ein halbtransparenter schwarzer Schatten nahe der unteren rechten Ecke – um das schöne, erhabene 3D-Aussehen der Box zu verstärken.

## Hinweise und Tipps

- Verwenden Sie den [W3C CSS Validator](https://jigsaw.w3.org/css-validator/), um Fehler in Ihrem CSS zu erkennen und zu beheben.
