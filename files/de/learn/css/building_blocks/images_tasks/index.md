## Aufgabe 2

In dieser Aufgabe haben Sie ein einfaches Formular. Ihre Aufgabe ist es, die folgenden Änderungen vorzunehmen:

- Verwenden Sie Attributselektoren, um das Suchfeld und die Schaltfläche innerhalb von `.myform` zu markieren.
- Sorgen Sie dafür, dass das Formularfeld und die Schaltfläche die gleiche Textgröße wie der Rest des Formulars verwenden.
- Geben Sie dem Formularfeld und der Schaltfläche 10 px Abstand.
- Geben Sie der Schaltfläche einen Hintergrund in `rebeccapurple`, eine weiße Vordergrundfarbe, keinen Rahmen und abgerundete Ecken von 5 px.
