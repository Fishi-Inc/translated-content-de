## Aufgabe 2

In dieser Aufgabe haben Sie ein einfaches Formular. Ihre Aufgabe ist es, folgende Änderungen vorzunehmen:

- Verwenden Sie Attributselektoren, um das Suchfeld und den Button innerhalb von `.myform` zu targeten.
- Passen Sie die Textgröße des Formularfeldes und des Buttons an die des restlichen Formulars an.
- Geben Sie dem Formularfeld und dem Button 10 px Padding.
- Geben Sie dem Button einen Hintergrund in `rebeccapurple`, eine weiße Vordergrundfarbe, keinen Rand und abgerundete Ecken von 5px.
