{{LearnSidebar}}

Ziel dieses Fähigkeitstests ist es, zu beurteilen, ob Sie das [Positionieren in CSS](/de/docs/Learn/CSS/CSS_layout/Positioning) mit der CSS-{{CSSxRef("position")}}-Eigenschaft und ihren Werten verstehen. Sie werden zwei kleine Aufgaben durchlaufen, die verschiedene Elemente des soeben behandelten Materials nutzen.

> [!NOTE]
> Sie können Lösungen in den interaktiven Editoren auf dieser Seite oder in einem Online-Editor wie [CodePen](https://codepen.io/), [JSFiddle](https://jsfiddle.net/) oder [Glitch](https://glitch.com/) ausprobieren.
>
> Wenn Sie nicht weiterkommen, können Sie sich in einem unserer [Kommunikationskanäle](/de/docs/MDN/Community/Communication_channels) an uns wenden.

In dieser Aufgabe sollen Sie das Element mit der Klasse `target` an die obere rechte Ecke des Containers positionieren, der einen grauen Rand von 5px hat.
