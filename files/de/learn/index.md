Willkommen im MDN-Lernbereich. Diese Artikelsammlung soll Anfängern der Webentwicklung grundlegende Fähigkeiten zum Codieren von Websites vermitteln. Ziel ist es nicht, Sie von "Anfänger" zu "Experte" zu machen, sondern von "Anfänger" zu "komfortabel". Von dort aus sollten Sie in der Lage sein, sich weiterzubilden, von [dem Rest von MDN](/en-US/) und anderen Ressourcen für Fortgeschrittene, die viel Vorwissen voraussetzen.

Wenn Sie ein völliger Anfänger sind, kann Webentwicklung herausfordernd sein — wir werden Ihnen zur Seite stehen und genügend Details bieten, damit Sie sich wohl fühlen und die Themen richtig erlernen können. Sie sollen sich wohlfühlen, egal ob Sie ein Student sind, der Webentwicklung lernt (alleine oder im Rahmen eines Kurses), ein Lehrer, der nach Unterrichtsmaterialien sucht, ein Hobbyist oder jemand, der einfach mehr darüber verstehen möchte, wie Webtechnologien funktionieren.

## Wo anfangen

Wenn Sie nicht sicher sind, ob Sie sich intensiv mit der Webentwicklung auseinandersetzen möchten und mit einer Kostprobe beginnen möchten, empfehlen wir Ihnen, mit unseren [Erste Schritte im Web](/de/docs/Learn/Getting_started_with_the_web) Leitfäden zu beginnen. Darüber hinaus sollten Sie folgende Themen lernen:

- HTML und CSS
  - : HTML bietet die Struktur für Webinhalte, während CSS Anweisungen zum Stylen und Layouten dieser Inhalte bietet. Siehe [Einführung in HTML](/de/docs/Learn/HTML/Introduction_to_HTML) und [CSS Erste Schritte](/de/docs/Learn/CSS/First_steps) für grundlegende Startinformationen.
- JavaScript
  - : JavaScript ermöglicht das Codieren interaktiver Funktionalität auf Websites. Beginnen Sie mit [JavaScript Erste Schritte](/de/docs/Learn/JavaScript/First_steps).
- Frameworks und Werkzeuge
  - : Nachdem Sie die Grundlagen von reinem HTML, CSS und JavaScript gemeistert haben, sollten Sie sich über [Client-seitige Webentwicklungstools](/de/docs/Learn/Tools_and_testing/Understanding_client-side_tools) informieren und dann in Betracht ziehen, sich mit [Client-seitigen JavaScript-Frameworks](/de/docs/Learn/Tools_and_testing/Client-side_JavaScript_frameworks) zu befassen. Sie sollten auch die grundlegenden Konzepte der [Server-seitigen Website-Programmierung](/de/docs/Learn/Server-side) erlernen.

> [!NOTE]
> Unser [Glossar](/de/docs/Glossary) bietet Terminologiedefinitionen. Wenn Sie außerdem eine spezifische Frage zur Webentwicklung haben, könnte die [Häufige Fragen](/de/docs/Learn/Common_questions) Sektion etwas bieten, das Ihnen weiterhilft.

## Abgedeckte Themen

Im Folgenden finden Sie eine Liste aller Themen, die wir im MDN-Lernbereich abdecken.

- [Erste Schritte im Web](/de/docs/Learn/Getting_started_with_the_web)
  - : Bietet eine praktische Einführung in die Webentwicklung für absolute Anfänger.
- [HTML — Strukturieren des Webs](/de/docs/Learn/HTML)
  - : HTML ist die Sprache, die wir verwenden, um die verschiedenen Teile unseres Inhalts zu strukturieren und zu definieren, welche Bedeutung oder Zweck sie haben. Dieses Thema lehrt HTML im Detail.
- [CSS — Stylen des Webs](/de/docs/Learn/CSS)
  - : CSS ist die Sprache, die wir verwenden, um das Aussehen und das Layout unserer Webinhalte zu steuern, sowie um Verhalten wie Animationen hinzuzufügen. Dieses Thema bietet eine umfassende Abdeckung von CSS.
- [JavaScript — Dynamisches clientseitiges Scripting](/de/docs/Learn/JavaScript)
  - : JavaScript ist die Skriptsprache, die verwendet wird, um dynamische Funktionalität zu Webseiten hinzuzufügen. Dieses Thema lehrt alle wesentlichen Aspekte, die benötigt werden, um sich im Schreiben und Verstehen von JavaScript wohlzufühlen.
- [Webformulare — Arbeiten mit Benutzerdaten](/de/docs/Learn/Forms)
  - : Webformulare sind ein mächtiges Werkzeug zur Interaktion mit Benutzern — am häufigsten werden sie zur Sammlung von Daten von Benutzern oder zur Steuerung einer Benutzeroberfläche genutzt. In den unten aufgeführten Artikeln werden wir alle wesentlichen Aspekte des Strukturierens, Stylens und Interagierens mit Webformularen abdecken.
- [Barrierefreiheit — das Web für alle nutzbar machen](/de/docs/Learn/Accessibility)
  - : Barrierefreiheit ist die Praxis, Webinhalte so zugänglich wie möglich für so viele Menschen wie möglich zu machen, unabhängig von Behinderung, Gerät, Standort oder anderen Unterscheidungsmerkmalen. Dieses Thema gibt Ihnen alles, was Sie wissen müssen.
- [Webleistung — Websites schnell und reaktionsfähig machen](/de/docs/Learn/Performance)
  - : Webleistung ist die Kunst, sicherzustellen, dass Webanwendungen schnell herunterladen und auf Benutzerinteraktionen reagieren, unabhängig von der Bandbreite, Bildschirmgröße, dem Netzwerk oder den Geräteeigenschaften des Benutzers.
- [MathML](/de/docs/Learn/MathML)
  - : MathML ist die Sprache, die wir verwenden können, um mathematische Formeln in Webseiten unter Verwendung von Brüchen, Schriftarten, Radikalen, Matrizen, Integralen, Serien usw. zu schreiben. Dieses Thema behandelt MathML.
- [Werkzeuge und Tests](/de/docs/Learn/Tools_and_testing)
  - : Dieses Thema behandelt die Werkzeuge, die Entwickler verwenden, um ihre Arbeit zu erleichtern, wie Cross-Browser-Testing-Tools, Linter, Formatierer, Transformationstools, Versionskontrollsysteme, Bereitstellungswerkzeuge und clientseitige JavaScript-Frameworks.
- [Server-seitige Website-Programmierung](/de/docs/Learn/Server-side)
  - : Selbst wenn Sie sich auf die clientseitige Webentwicklung konzentrieren, ist es dennoch nützlich zu wissen, wie Server und serverseitige Codefunktionen funktionieren. Dieses Thema bietet eine allgemeine Einführung in die Funktionsweise der Serverseite und detaillierte Anleitungen, wie man eine serverseitige App mithilfe von zwei beliebten Frameworks erstellt: Django (Python) und Express (Node.js).

## Unsere Codebeispiele abrufen

Die Codebeispiele, die Sie im Lernbereich finden werden, sind alle [auf GitHub verfügbar](https://github.com/mdn/learning-area/).
Wenn Sie sie alle auf Ihren Computer kopieren möchten, ist der einfachste Weg, [ein ZIP der neuesten Haupt-Codezweig herunterzuladen](https://codeload.github.com/mdn/learning-area/zip/main).

Wenn Sie das Repository auf eine flexiblere Weise kopieren möchten, die automatische Updates ermöglicht, können Sie den komplexeren Anweisungen folgen:

1. [Installieren Sie Git](https://git-scm.com/downloads) auf Ihrem Rechner. Dies ist die zugrunde liegende Versionskontrollsystem-Software, auf der GitHub basiert.
2. Öffnen Sie die [Befehlszeile](https://www.lifewire.com/how-to-open-command-prompt-2618089) Ihres Computers (Windows) oder das Terminal ([Linux](https://help.ubuntu.com/community/UsingTheTerminal), [macOS](https://blog.teamtreehouse.com/introduction-to-the-mac-os-x-command-line)).
3. Verwenden Sie den folgenden Befehl, um das Lernbereich-Repo in einen Ordner namens "learning-area" am aktuellen Speicherort Ihres Eingabeaufforderungs-/Terminal-Pointers zu kopieren:

   ```bash
   git clone https://github.com/mdn/learning-area
   ```

4. Sie können jetzt das Verzeichnis betreten und die Dateien finden, die Sie suchen (entweder mit Hilfe Ihres Finders/File Explorers oder des [`cd`-Befehls](<https://en.wikipedia.org/wiki/Cd_(command)>)).

Sie können das `learning-area` Repository mit allen Änderungen an der Hauptversion auf GitHub mit den folgenden Schritten aktualisieren:

1. Gehen Sie in Ihrer Eingabeaufforderung/Ihrem Terminal in das `learning-area` Verzeichnis mit `cd`. Zum Beispiel, wenn Sie sich im übergeordneten Verzeichnis befanden:

   ```bash
   cd learning-area
   ```

2. Aktualisieren Sie das Repository mit dem folgenden Befehl:

   ```bash
   git pull
   ```
