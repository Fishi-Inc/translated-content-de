## Grundlegende Steuerelemente 1

Diese Aufgabe beginnt sanft, indem Sie zwei `<input>`-Elemente für die Benutzer-ID und das Passwort erstellen, zusammen mit einem Absenden-Button.

1. Erstellen Sie geeignete Eingaben für Benutzer-ID und Passwort.
2. Sie sollten diese auch mit ihren Textetiketten semantisch verknüpfen.
3. Erstellen Sie einen Absenden-Button im verbleibenden Listenelement mit dem Button-Text "Log in".

## Grundlegende Steuerelemente 2

Die nächste Aufgabe erfordert es, funktionierende Sets von Checkboxen und Optionsfeldern aus den bereitgestellten Textelementen zu erstellen.

1. Wandeln Sie den Inhalt des ersten `<fieldset>` in eine Gruppe von Optionsfeldern um — es sollte nur möglich sein, einen Pony-Charakter gleichzeitig auszuwählen.
2. Stellen Sie sicher, dass das erste Optionsfeld beim Laden der Seite ausgewählt ist.
3. Wandeln Sie den Inhalt des zweiten `<fieldset>` in eine Gruppe von Checkboxen um.
4. Fügen Sie ein paar weitere Hotdog-Optionen Ihrer Wahl hinzu.

## Grundlegende Steuerelemente 3

Die letzte Aufgabe in diesem Set erfordert es, einen Dateiauswahl-Dialog zu erstellen.

1. Erstellen Sie einen grundlegenden Dateiauswahl-Dialog.
2. Erlauben Sie dem Benutzer, mehrere Dateien gleichzeitig auszuwählen.
3. Erlauben Sie, dass der Dateiauswahl-Dialog nur JPG- und PNG-Bilder akzeptiert.
