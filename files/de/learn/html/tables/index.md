Eine sehr häufige Aufgabe in HTML ist das Strukturieren von tabellarischen Daten, und es gibt eine Reihe von Elementen und Attributen speziell für diesen Zweck. Gepaart mit ein wenig [CSS](/de/docs/Learn/CSS) für das Styling, macht es HTML einfach, Tabellen mit Informationen im Web darzustellen, wie zum Beispiel Ihren Stundenplan, den Zeitplan in Ihrem örtlichen Schwimmbad oder Statistiken über Ihre Lieblingsdinosaurier oder Ihr Lieblingsteam. Dieses Modul führt Sie durch alles, was Sie über das Strukturieren von tabellarischen Daten mit HTML wissen müssen.

## Voraussetzungen

Bevor Sie mit diesem Modul beginnen, sollten Sie bereits die Grundlagen von HTML behandelt haben — siehe [Einführung in HTML](/de/docs/Learn/HTML/Introduction_to_HTML).

> [!NOTE]
> Wenn Sie an einem Computer/Tablet/anderen Gerät arbeiten, auf dem Sie nicht in der Lage sind, eigene Dateien zu erstellen, können Sie (die meisten) der Codebeispiele in einem Online-Coding-Programm wie [JSBin](https://jsbin.com/) oder [Glitch](https://glitch.com/) ausprobieren.

## Leitfäden

Dieses Modul enthält die folgenden Artikel, die Sie durch alle Grundlagen zur Erstellung von Tabellen in HTML führen.

- [Grundlagen von HTML-Tabellen](/de/docs/Learn/HTML/Tables/Basics)
  - : In diesem Artikel beginnen Sie mit HTML-Tabellen und behandeln die Grundlagen wie Zeilen und Zellen, Überschriften, das Spannen von Zellen über mehrere Spalten und Zeilen und wie man alle Zellen in einer Spalte für Styling-Zwecke zusammenfasst.
- [Erweiterte Funktionen und Zugänglichkeit von HTML-Tabellen](/de/docs/Learn/HTML/Tables/Advanced)
  - : Dieser Artikel behandelt einige fortgeschrittenere Funktionen von HTML-Tabellen — wie Bildunterschriften/Zusammenfassungen und das Gruppieren Ihrer Zeilen in Tabellenkopf-, Körper- und Fußbereiche — sowie die Betrachtung der Zugänglichkeit von Tabellen für sehbehinderte Benutzer.
