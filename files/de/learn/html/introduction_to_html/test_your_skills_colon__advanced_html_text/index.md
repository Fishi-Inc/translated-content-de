## Aufgabe 2

In dieser Aufgabe möchten wir, dass Sie der bereitgestellten HTML-Datei einige semantische Merkmale hinzufügen:

- Verwandeln Sie den zweiten Absatz in ein Block-Zitat und zeigen Sie semantisch an, dass das Zitat aus [Accessibility](/de/docs/Learn/Accessibility) stammt.
- Markieren Sie "HTML" und "CSS" semantisch als Akronyme und geben Sie Erweiterungen als Tooltips an.
- Verwenden Sie Tief- und Hochgestellt, um korrekte Semantik für die chemischen Formeln und Daten bereitzustellen und sie korrekt anzuzeigen.
- Semantisch verknüpfen Sie maschinenlesbare Daten mit den Daten im Text.
