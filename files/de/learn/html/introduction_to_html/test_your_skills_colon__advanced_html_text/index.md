## Aufgabe 2

In dieser Aufgabe möchten wir, dass Sie einige Semantiken wie folgt zum bereitgestellten HTML hinzufügen:

- Verwandeln Sie den zweiten Absatz in ein Blockquote und kennzeichnen Sie semantisch, dass das Zitat aus [Accessibility](/de/docs/Learn/Accessibility) stammt.
- Kennzeichnen Sie "HTML" und "CSS" semantisch als Akronyme und bieten Sie Erweiterungen als Tooltips an.
- Verwenden Sie Subscript und Superscript, um die korrekten Semantiken für die chemischen Formeln und Daten bereitzustellen und sie korrekt anzuzeigen.
- Verknüpfen Sie semantisch maschinenlesbare Daten mit den Daten im Text.
