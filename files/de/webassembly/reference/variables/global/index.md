Die **`global`**-Instruktion deklariert eine neue globale Variable.

{{EmbedInteractiveExample("pages/wat/global.html", "tabbed-taller")}}

## Syntax

```wasm
;; declare new variable named $val of type i32
(global $val i32)
```
