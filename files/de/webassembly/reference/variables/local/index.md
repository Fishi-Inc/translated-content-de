Die **`local`**-Anweisung deklariert eine neue lokale Variable.

{{EmbedInteractiveExample("pages/wat/local.html", "tabbed-taller")}}

## Syntax

```wasm
;; declare new variable named $val of type i32
(local $val i32)
```
