**DNS** (Domain Name System) ist ein hierarchisches und dezentralisiertes Benennungssystem für internetverbundene Ressourcen. DNS verwaltet eine Liste von {{Glossary("domain name","domain names")}} zusammen mit den Ressourcen, wie z.B. IP-Adressen, die mit ihnen verbunden sind.

Die bekannteste Funktion von DNS ist die Übersetzung von benutzerfreundlichen Domainnamen (wie mozilla.org) in eine numerische {{Glossary("IP address")}} (wie 192.0.2.172); dieser Prozess der Zuordnung eines Domainnamens zur entsprechenden IP-Adresse wird als **DNS-Abfrage** bezeichnet. Im Gegensatz dazu wird eine **Reverse-DNS-Abfrage** (rDNS) verwendet, um den mit einer IP-Adresse verknüpften Domainnamen zu ermitteln.
