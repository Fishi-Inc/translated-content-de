_Scalable Vector Graphics_ (**SVG**) ist ein 2D-Vektorbildformat, das auf einer [XML](/de/docs/Glossary/XML)-Syntax basiert.

Das [W3C](/de/docs/Glossary/W3C) begann in den späten 1990er Jahren mit der Arbeit an SVG, aber SVG wurde erst populär, als der [Internet Explorer](/de/docs/Glossary/Microsoft_Internet_Explorer) 9 mit SVG-Unterstützung herauskam. Alle wichtigen [Browser](/de/docs/Glossary/browser) unterstützen jetzt SVG.

Basierend auf einer [XML](/de/docs/Glossary/XML)-Syntax kann SVG mit [CSS](/de/docs/Glossary/CSS) gestaltet und interaktiv mit [JavaScript](/de/docs/Glossary/JavaScript) gemacht werden. HTML ermöglicht das direkte Einbetten von SVG-[Tags](/de/docs/Glossary/Tag) in ein [HTML](/de/docs/Glossary/HTML)-Dokument.

Als [Vektorbildformat](https://en.wikipedia.org/wiki/Vector_graphics) können SVG-Grafiken unendlich skaliert werden, was sie im [responsiven Design](/de/docs/Glossary/responsive_web_design) unverzichtbar macht, da Sie Benutzerschnittstellenelemente und Grafiken erstellen können, die sich an jede Bildschirmgröße anpassen. SVG bietet auch eine nützliche Reihe von Werkzeugen wie Clipping, Maskierung, Filter und Animationen.
