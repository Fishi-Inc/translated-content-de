Eine **Firewall** ist ein System, das den Netzwerkverkehr filtert. Es kann ihn entweder passieren lassen oder blockieren, entsprechend einiger festgelegter Regeln. Zum Beispiel kann es eingehende Verbindungen, die auf einen bestimmten Port abzielen, oder ausgehende Verbindungen zu einer bestimmten IP-Adresse blockieren.
