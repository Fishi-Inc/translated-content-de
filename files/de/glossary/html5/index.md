Der Begriff HTML5 ist im Wesentlichen ein Schlagwort, das sich auf eine Reihe moderner Webtechnologien bezieht. Dies umfasst den {{Glossary("HTML")}} Living Standard sowie {{glossary("JavaScript")}} {{glossary("API","APIs")}}, um Speicher, Multimedia und Hardware-Zugriff zu verbessern.

Sie werden manchmal von "neuen HTML5-Elementen" hören oder finden HTML5 als eine neue Version von HTML beschrieben. HTML5 war der Nachfolger der vorherigen HTML-Versionen und führte neue Elemente und Fähigkeiten zur Sprache ein, basierend auf der vorherigen Version HTML 4.01, und verbesserte oder entfernte einige bestehende Funktionen. Da HTML jedoch ein Living Standard ist, hat es jetzt keine Versionen mehr. Die aktuelle Spezifikation finden Sie unter [html.spec.whatwg.org/](https://html.spec.whatwg.org/).

Jede moderne Website sollte den [HTML-Doctype](/de/docs/MDN/Writing_guidelines/Writing_style_guide/Code_style_guide/HTML#doctype) verwenden — dies stellt sicher, dass Sie die neueste Version von HTML verwenden.

> [!NOTE]
> Bis 2019 veröffentlichte das {{glossary("W3C")}} einen konkurrierenden HTML5-Standard mit Versionsnummern. Seit dem [28. Mai 2019](https://www.w3.org/blog/news/archives/7753) wurde der WHATWG Living Standard vom W3C als die einzige Version von HTML bekannt gegeben.
