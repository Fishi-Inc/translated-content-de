In JavaScript können Objekte als eine Sammlung von Eigenschaften betrachtet werden. Mit der [Objektliteral-Syntax](/de/docs/Web/JavaScript/Guide/Grammar_and_types#object_literals) wird eine begrenzte Menge von Eigenschaften initialisiert; anschließend können Eigenschaften hinzugefügt und entfernt werden. Eigenschaftenwerte können Werte beliebigen Typs, einschließlich anderer Objekte, sein, was es ermöglicht, komplexe Datenstrukturen aufzubauen. Eigenschaften werden mit _Schlüssel_-Werten identifiziert. Ein _Schlüssel_-Wert ist entweder ein [String-Wert](/de/docs/Glossary/String) oder ein [Symbol-Wert](/de/docs/Web/JavaScript/Reference/Global_Objects/Symbol).

Es gibt zwei Arten von Objekteigenschaften: Die [_Dateneigenschaft_](/de/docs/Web/JavaScript/Data_structures#data_property) und die [_Zugriffseigenschaft_](/de/docs/Web/JavaScript/Data_structures#accessor_property).

> [!NOTE]
> Es ist wichtig zu erkennen, dass es sich um eine Zugriffseigenschaft handelt — nicht um eine Zugriffsmethode. Wir können einem JavaScript-Objekt klassenähnliche Zugriffe geben, indem wir eine Funktion als Wert verwenden — aber das macht das Objekt nicht zu einer Klasse.
