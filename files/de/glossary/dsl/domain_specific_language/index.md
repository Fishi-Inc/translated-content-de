Eine **Domänenspezifische Sprache (DSL)** ist eine Art von Programmiersprache mit begrenztem Umfang, die entwickelt wurde, um ein bestimmtes Problem innerhalb eines Anwendungsbereichs zu adressieren.

Stellen Sie DSLs den _allzwecksprachen (GPLs)_ gegenüber, die entwickelt wurden, um verschiedene Probleme über verschiedene Domänen hinweg zu lösen.
