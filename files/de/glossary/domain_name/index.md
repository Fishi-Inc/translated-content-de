Ein **Domainname** ist die Adresse einer Website im {{Glossary("Internet")}}. Domainnamen werden in {{Glossary("URL","URLs")}} verwendet, um zu identifizieren, zu welchem Server eine bestimmte Webseite gehört. Der Domainname besteht aus einer hierarchischen Abfolge von Namen (Labels), die durch Punkte getrennt sind und mit einer {{glossary("TLD","Erweiterung")}} enden.
