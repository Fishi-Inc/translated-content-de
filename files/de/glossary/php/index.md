## Beispiele

### Grundsyntax

```php
// start of PHP code
<?php
  // PHP code goes here
?>
// end of PHP code
```

### Daten auf dem Bildschirm ausgeben

```php
<?php
  echo "Hello World!";
?>
```

### PHP-Variablen

```php
<?php
  // variables
  $name='Danilo';
  $surname='Santos';
  $country='Brasil';
  $email='danilocarsan@gmail.com';

  // printing the variables
  echo $name;
  echo $surname;
  echo $country;
  echo $email;
?>
```
