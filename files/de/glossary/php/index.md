## Beispiele

### Grundlegende Syntax

```php
// Start des PHP-Codes
<?php
  // PHP-Code geht hierhin
?>
// Ende des PHP-Codes
```

### Daten auf dem Bildschirm ausgeben

```php
<?php
  echo "Hello World!";
?>
```

### PHP-Variablen

```php
<?php
  // Variablen
  $nome='Danilo';
  $sobrenome='Santos';
  $pais='Brasil';
  $email='danilocarsan@gmail.com';

  // Ausgeben der Variablen
  echo $nome;
  echo $sobrenome;
  echo $pais;
  echo $email;
?>
```
