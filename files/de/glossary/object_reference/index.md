Eine **Objektreferenz** ist eine Verknüpfung zu einem _{{glossary("object")}}_. Objektreferenzen können genau wie die verknüpften Objekte verwendet werden.

Das Konzept der Objektreferenzen wird deutlich, wenn dasselbe Objekt mehr als einer _{{Glossary("property/javascript", "Eigenschaft")}}_ zugewiesen wird. Anstatt eine Kopie des Objekts zu halten, enthält jede zugewiesene Eigenschaft Objektreferenzen, die auf dasselbe Objekt verweisen, sodass, wenn sich das Objekt ändert, alle Eigenschaften, die auf das Objekt verweisen, die Änderung widerspiegeln.
