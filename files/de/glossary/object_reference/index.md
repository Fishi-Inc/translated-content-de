Ein **Objektverweis** ist eine Verknüpfung zu einem _[Objekt](/de/docs/Glossary/object)_. Objektverweise können genau wie die verknüpften Objekte verwendet werden.

Das Konzept der Objektverweise wird deutlich, wenn dasselbe Objekt mehreren _[Eigenschaften](/de/docs/Glossary/property/javascript)_ zugewiesen wird. Anstatt eine Kopie des Objekts zu halten, hält jede zugewiesene Eigenschaft Objektverweise, die auf dasselbe Objekt verweisen, sodass bei Änderungen des Objekts alle dieses Objekt referenzierenden Eigenschaften die Änderung widerspiegeln.
