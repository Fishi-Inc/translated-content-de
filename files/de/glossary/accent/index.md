Ein **Akzent** ist eine typischerweise helle Farbe, die im Kontrast zu den eher zweckmäßigen Hintergrund- und Vordergrundfarben innerhalb eines Farbschemas steht. Diese sind in der visuellen Gestaltung vieler Plattformen (aber nicht aller) vorhanden.

Im Web wird ein Akzent manchmal in `{{HTMLElement("input")}}`-Elementen für den aktiven Teil des Steuerelements verwendet, zum Beispiel der Hintergrund eines aktivierten [Checkbox](/de/docs/Web/HTML/Element/input/checkbox).

### CSS im Zusammenhang mit dem Akzent

Sie können die Farbe des Akzents für ein bestimmtes Element festlegen, indem Sie die CSS-Eigenschaft `{{cssxref("accent-color")}}` des Elements auf den entsprechenden `{{cssxref("&lt;color&gt;")}}`-Wert setzen.
