Flexbox ist der gebräuchliche Name für das [CSS flexible Box-Layout-Modul](/de/docs/Web/CSS/CSS_flexible_box_layout), ein Layoutmodell für die Anzeige von Elementen in einer Dimension – als Reihe oder als Spalte.

In der Spezifikation wird Flexbox als ein Layoutmodell für das Design von Benutzeroberflächen beschrieben. Das Hauptmerkmal von Flexbox ist, dass sich Elemente in einem Flex-Layout vergrößern und verkleinern können. Der Raum kann den Elementen selbst zugewiesen oder zwischen oder um die Elemente herum verteilt werden.

Flexbox ermöglicht auch die Ausrichtung von Elementen auf der Haupt- oder Querachse und bietet so ein hohes Maß an Kontrolle über die Größe und Ausrichtung einer Gruppe von Elementen.
