Flexbox ist der gebräuchliche Name für das [CSS flexible box layout module](/de/docs/Web/CSS/CSS_flexible_box_layout), ein Layoutmodell zur Anzeige von Elementen in einer einzigen Dimension — als Reihe oder Spalte.

In der Spezifikation wird Flexbox als Layoutmodell für das Design von Benutzeroberflächen beschrieben. Das Hauptmerkmal von Flexbox ist, dass Elemente in einem Flex-Layout wachsen und schrumpfen können. Der Platz kann den Elementen selbst zugewiesen oder zwischen oder um die Elemente herum verteilt werden.

Flexbox ermöglicht auch die Ausrichtung von Elementen auf der Haupt- oder Kreuzachse und bietet somit ein hohes Maß an Kontrolle über die Größe und Ausrichtung einer Gruppe von Elementen.
