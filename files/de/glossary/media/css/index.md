Im Kontext von [CSS](/de/docs/Glossary/CSS) (Cascading Style Sheets) bezieht sich der Begriff **_Medien_** auf das Ziel, auf das das Dokument vom [Rendering-Engine](/de/docs/Glossary/Engine/Rendering) gezeichnet werden soll.

Typischerweise ist dies ein Bildschirm, es kann aber auch ein Drucker, eine Braille-Anzeige oder ein anderer Gerätetyp sein.

CSS bietet mehrere Funktionen, die es Ihnen ermöglichen, die Stile Ihres Dokuments anzupassen oder sogar verschiedene Stile anzubieten, abhängig vom Medientyp (wie Bildschirm oder Druck) oder den Medienfähigkeiten (wie Breite, Auflösung oder anderen Werten) des Geräts des Betrachters.
