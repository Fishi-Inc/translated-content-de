**Snake Case** ist eine Methode, um Phrasen ohne Leerzeichen zu schreiben, wobei Leerzeichen durch Unterstriche `_` ersetzt werden und die Wörter typischerweise alle in Kleinbuchstaben geschrieben sind. Es wird oft als "snake_case" stilisiert, um den Leser an sein Erscheinungsbild zu erinnern.

Snake Case wird oft als Konvention zur Variablennamensgebung verwendet. Die folgenden Namen sind im Snake Case: `left_shift`, `bitwise_invert`, `matrix_transpose`.

Beachten Sie, dass Snake Case keine Großbuchstaben enthält. Manchmal werden Konstanten vollständig in Großbuchstaben geschrieben, wie zum Beispiel JavaScripts {{jsxref("Number.MAX_SAFE_INTEGER")}}. Dies wird normalerweise nicht als Snake Case betrachtet. Stattdessen wird es manchmal als _Screaming Snake Case_ bezeichnet.

Snake Case ist die populärste Konvention in Python, Rust und verschiedenen anderen Sprachen.
