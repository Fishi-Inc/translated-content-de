**Snake case** ist eine Schreibweise für Phrasen ohne Leerzeichen, bei der Leerzeichen durch Unterstriche `_` ersetzt werden und die Wörter typischerweise kleingeschrieben sind. Es wird oft als "snake_case" stilisiert, um den Leser an sein Erscheinungsbild zu erinnern.

Snake Casing wird häufig als Konvention zur Benennung von Variablen verwendet. Die folgenden Namen sind in Snake Case: `left_shift`, `bitwise_invert`, `matrix_transpose`.

Beachten Sie, dass Snake Case niemals Großbuchstaben enthält. Manchmal werden Konstanten komplett in Großbuchstaben geschrieben, wie zum Beispiel JavaScripts {{jsxref("Number.MAX_SAFE_INTEGER")}}. Dies wird normalerweise nicht als Snake Case betrachtet. Stattdessen wird es manchmal _Screaming Snake Case_ genannt.

Snake Case ist die beliebteste Konvention in Python, Rust und verschiedenen anderen Sprachen.
