Ein **Symbol** ist ein Datentyp, der einzigartige, unveränderliche Kennungen darstellt. Sie werden manchmal _Atoms_ genannt.

Da ein Symbol einzigartig und unveränderlich ist, können Sie einen Eigenschaftswert, der mit einem Symbol verknüpft ist, nur lesen, wenn Sie einen Verweis auf die ursprüngliche Kennung haben.

In JavaScript ist `symbol` einer der {{Glossary("primitive", "primitiven Typen")}} und kann mit der [`Symbol()`](/de/docs/Web/JavaScript/Reference/Global_Objects/Symbol) Fabrikfunktion erstellt werden, die jedes Mal ein anderes Symbol zurückgibt. Sie können als Schlüssel für Objekte verwendet werden, die niemals versehentlich mit anderen Eigenschaften kollidieren können.

JavaScript definiert auch zwei andere Kategorien von Symbolen: wohlbekannte Symbole und registrierte Symbole. Lesen Sie die {{jsxref("Symbol")}}-Referenz für weitere Informationen.
