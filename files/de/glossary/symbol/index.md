Ein **Symbol** ist ein Datentyp, der eindeutige, unverfälschliche Bezeichner darstellt. Sie werden manchmal auch _Atome_ genannt.

Da ein Symbol eindeutig und unverfälschlich ist, können Sie einen mit einem Symbol verknüpften Eigenschaftswert nur lesen, wenn Sie einen Verweis auf den ursprünglichen Bezeichner haben.

In JavaScript ist `symbol` einer der [primitiven Typen](/de/docs/Glossary/primitive) und kann mit der [`Symbol()`](/de/docs/Web/JavaScript/Reference/Global_Objects/Symbol) Fabrikmethode erstellt werden, die jedes Mal ein anderes Symbol zurückgibt. Sie können als Schlüssel für Objekte verwendet werden, die niemals versehentlich mit anderen Eigenschaften kollidieren.

JavaScript definiert auch zwei andere Kategorien von Symbolen: wohlbekannte Symbole und registrierte Symbole. Lesen Sie die {{jsxref("Symbol")}} Referenz für weitere Informationen.
