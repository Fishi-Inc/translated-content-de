**WebRTC** (_Web Real-Time Communication_) ist eine [API](/de/docs/Glossary/API), die von Web-Apps für Video-Chat, Sprach-Anruf und P2P-Dateifreigabe verwendet werden kann.

WebRTC besteht hauptsächlich aus diesen Teilen:

- [`getUserMedia()`](/de/docs/Web/API/MediaDevices/getUserMedia)
  - : Gewährt Zugriff auf die Kamera und/oder das Mikrofon eines Geräts und kann deren Signale in eine RTC-Verbindung einfügen.
- [`RTCPeerConnection`](/de/docs/Web/API/RTCPeerConnection)
  - : Eine Schnittstelle zur Konfiguration von Video- oder Sprachanrufen.
- [`RTCDataChannel`](/de/docs/Web/API/RTCDataChannel)
  - : Bietet eine Methode zur Einrichtung eines [peer-to-peer](/de/docs/Glossary/P2P) Datenpfads zwischen Browsern.
