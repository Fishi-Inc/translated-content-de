Internet Explorer (oder IE) war ein kostenloser grafischer {{glossary("browser")}}, der von Microsoft für ältere Unternehmensanwendungen gewartet wurde. {{glossary("Microsoft Edge")}} ist derzeit der Standardbrowser für Windows.

Microsoft lieferte IE erstmals 1995 mit Windows als Teil des Pakets "Microsoft Plus!" aus. Bis etwa 2002 war Internet Explorer der meistgenutzte Browser der Welt, verlor jedoch an Boden gegenüber Chrome, Firefox, Edge und Safari.

IE durchlief viele Versionen und stellte Versionen für Desktop, Mobilgeräte und Xbox-Konsole bereit. Es war auch auf Mac und UNIX verfügbar, allerdings stellte Microsoft diese Versionen 2003 bzw. 2001 ein. Die letzte Windows-Version war Windows 11.0.220 am 10. November 2020.

Microsoft beendete den Support für IE am 15. Juni 2022.
