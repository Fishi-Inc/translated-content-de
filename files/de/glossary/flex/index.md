{{GlossarySidebar}}

`flex` ist ein Wert der CSS-{{cssxref("display")}}-Eigenschaft. Zusammen mit `inline-flex` bewirkt es, dass das Element, auf das es angewendet wird, zu einem {{glossary("flex container")}} wird, und die Kinder des Elements jeweils zu einem {{glossary("flex item")}} werden. Die Elemente nehmen dann am Flex-Layout teil, und alle in dem [CSS Flexible Box Layout Modul](/de/docs/Web/CSS/CSS_flexible_box_layout) definierten Eigenschaften können angewendet werden.

Es gibt auch eine {{cssxref("flex")}}-Eigenschaft, die eine Kurzform für die Flexbox-Eigenschaften {{cssxref("flex-grow")}}, {{cssxref("flex-shrink")}} und {{cssxref("flex-basis")}} ist. Diese Eigenschaft ist nur auf Flex-Container anwendbar.

Zusätzlich kann [`<flex>`](/de/docs/Web/CSS/flex_value) sich auf eine [flexible Länge](/de/docs/Web/CSS/flex_value) im CSS Grid Layout beziehen.
