{{GlossarySidebar}}

`flex` ist ein Wert der CSS-{{cssxref("display")}}-Eigenschaft. Zusammen mit `inline-flex` bewirkt es, dass das Element, auf das es angewendet wird, zu einem [flex container](/de/docs/Glossary/flex_container) wird und die Kinder dieses Elements jeweils zu einem [flex item](/de/docs/Glossary/flex_item) werden. Die Elemente nehmen dann am Flex-Layout teil, und alle im [CSS-Flexible-Box-Layout-Modul](/de/docs/Web/CSS/CSS_flexible_box_layout) definierten Eigenschaften können angewendet werden.

Es gibt auch eine {{cssxref("flex")}}-Eigenschaft, die eine Kurzform für die Flexbox-Eigenschaften {{cssxref("flex-grow")}}, {{cssxref("flex-shrink")}} und {{cssxref("flex-basis")}} darstellt. Diese Eigenschaft ist nur auf Flex-Container anwendbar.

Darüber hinaus kann [`<flex>`](/de/docs/Web/CSS/flex_value) in CSS Grid Layout eine [flexible Länge](/de/docs/Web/CSS/flex_value) darstellen.
