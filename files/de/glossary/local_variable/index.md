{{GlossarySidebar}}

Eine {{glossary("variable")}}, deren Name nur innerhalb eines {{Glossary("local scope")}} an ihren {{glossary("value")}} gebunden ist.

## Beispiel

```js
let global = 5; // Eine globale Variable

function fun() {
  let local = 10; // Eine lokale Variable
}
```
