{{GlossarySidebar}}

Die **Table Wrapper Box** ist die Box, die um {{Glossary("table grid box", "table grid boxes")}} erstellt wird und den benötigten Platz für jede für die Tabelle angezeigte Überschrift berücksichtigt. Diese Box wird zum enthaltenden Block für absolut positionierte Elemente, bei denen die Tabelle der enthaltende Block ist.
