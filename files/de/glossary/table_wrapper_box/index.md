{{GlossarySidebar}}

Die **Table Wrapper Box** ist die Box, die um [table grid boxes](/de/docs/Glossary/table_grid_box) herum erzeugt wird und den Platz berücksichtigt, der für eine eventuell angezeigte Beschriftung der Tabelle benötigt wird. Diese Box wird der enthaltende Block für absolut positionierte Elemente, bei denen die Tabelle der enthaltende Block ist.
