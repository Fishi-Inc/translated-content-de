Type conversion (oder Typumwandlung) bedeutet die Übertragung von Daten von einem Datentyp in einen anderen. Eine _implizite Umwandlung_ erfolgt, wenn der Compiler (bei kompilierten Sprachen) oder die Laufzeitumgebung (bei Skriptsprachen wie [JavaScript](/de/docs/Glossary/JavaScript)) Datentypen automatisch umwandelt. Der Quellcode kann auch _explizit_ eine Umwandlung erfordern.

Beispielsweise wird in dem Ausdruck `"foo" + 1` die [Number](/de/docs/Glossary/Number) `1` implizit in einen [String](/de/docs/Glossary/String) umgewandelt und der Ausdruck gibt `"foo1"` zurück. Bei der Anweisung `Number("0x11")` wird der String `"0x11"` explizit in die Zahl `17` umgewandelt.
