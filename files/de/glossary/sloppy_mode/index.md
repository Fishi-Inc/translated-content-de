[ECMAScript](/de/docs/Glossary/ECMAScript) 5 und später ermöglichen es Skripten, sich für einen neuen [strict mode](/de/docs/Web/JavaScript/Reference/Strict_mode) zu entscheiden, der die Semantik von JavaScript auf verschiedene Weise ändert, um die Widerstandsfähigkeit zu verbessern und um es einfacher zu machen, zu verstehen, was passiert, wenn es Probleme gibt.

Der normale, nicht-strikte Modus von JavaScript wird manchmal als **sloppy mode** bezeichnet. Dies ist keine offizielle Bezeichnung, aber Sie werden wahrscheinlich darauf stoßen, wenn Sie ernsthaft mit JavaScript-Code arbeiten.
