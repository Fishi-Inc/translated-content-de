Ein **CSS-Vorprozessor** ist ein Programm, mit dem Sie [CSS](/de/docs/Glossary/CSS) aus der eigenen einzigartigen [Syntax](/de/docs/Glossary/syntax) des Vorprozessors generieren können.

Es gibt viele CSS-Vorprozessoren zur Auswahl, jedoch werden die meisten CSS-Vorprozessoren einige Funktionen hinzufügen, die in reinem CSS nicht existieren, wie Mixin, verschachtelte Selektoren, Vererbung von Selektoren und so weiter. Diese Funktionen machen die CSS-Struktur lesbarer und einfacher zu warten.

Um einen CSS-Vorprozessor zu verwenden, müssen Sie einen CSS-Compiler auf Ihrem Web-[Server](/de/docs/Glossary/server) installieren; Oder verwenden Sie den CSS-Vorprozessor, um in der Entwicklungsumgebung zu kompilieren, und laden Sie dann die kompilierte CSS-Datei auf den Webserver hoch.
