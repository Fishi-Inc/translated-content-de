Der **Layout-Viewport** ist das Ansichtsfenster, in dem der Browser eine Webseite zeichnet. Es repräsentiert im Wesentlichen das, was sichtbar sein kann, während der [visuelle Viewport](/de/docs/Glossary/visual_viewport) das darstellt, was aktuell auf dem Anzeigegerät des Benutzers sichtbar ist.

Dies wird beispielsweise auf mobilen Geräten wichtig, auf denen man normalerweise durch eine Kneifgeste in die Inhalte einer Website hinein- oder herauszoomen kann. Das gerenderte Dokument ändert sich dabei in keiner Weise, sodass der Layout-Viewport unverändert bleibt, während der Benutzer die Zoomstufe anpasst. Stattdessen wird der visuelle Viewport aktualisiert, um den Bereich der Seite anzuzeigen, den sie sehen können.
