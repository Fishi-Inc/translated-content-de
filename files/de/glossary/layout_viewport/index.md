Der **Layout-Viewport** ist der Viewport, in den der Browser eine Webseite zeichnet. Im Wesentlichen stellt er dar, was sichtbar ist, während der {{Glossary("visual viewport")}} das repräsentiert, was auf dem Anzeigegerät des Benutzers aktuell sichtbar ist.

Dies wird z. B. auf mobilen Geräten wichtig, wo mit einer Kneifgeste üblicherweise in den Inhalt einer Website hinein- oder herausgezoomt werden kann. Das gerenderte Dokument ändert sich in keiner Weise, daher bleibt der Layout-Viewport gleich, wenn der Benutzer die Zoomstufe anpasst. Stattdessen wird der visuelle Viewport aktualisiert, um den Bereich der Seite anzuzeigen, den der Benutzer sehen kann.
