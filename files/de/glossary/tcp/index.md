**TCP (Transmission Control Protocol)** ist ein wichtiges Netzwerk-{{Glossary("protocol")}}, das es zwei Hosts ermöglicht, eine Verbindung herzustellen und Datenströme auszutauschen. TCP garantiert die Zustellung von Daten und Paketen in der gleichen Reihenfolge, in der sie gesendet wurden. Vint Cerf und Bob Kahn, die damals Wissenschaftler bei DARPA waren, entwickelten TCP in den 1970er Jahren.

Die Aufgabe von TCP ist es, sicherzustellen, dass die Pakete zuverlässig und fehlerfrei geliefert werden. TCP implementiert [Staukontrolle](https://en.wikipedia.org/wiki/TCP_congestion_control), was bedeutet, dass die anfänglichen Anforderungen klein beginnen und sich schrittweise auf das Niveau der Bandbreite erhöhen, die die Computer, Server und das Netzwerk unterstützen können.
