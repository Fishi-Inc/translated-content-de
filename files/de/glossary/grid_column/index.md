Eine **Grid-Spalte** ist eine vertikale Spur in einem [CSS Grid Layout](/de/docs/Web/CSS/CSS_grid_layout) und der Raum zwischen zwei vertikalen Grid-Linien. Sie wird durch die {{cssxref("grid-template-columns")}} Eigenschaft oder in den Kurzformen {{cssxref("grid")}} oder {{cssxref("grid-template")}} definiert.

Zusätzlich können Spalten im _impliziten Grid_ erstellt werden, wenn Elemente außerhalb der im _expliziten Grid_ erstellten Spalten platziert werden. Diese Spalten werden standardmäßig automatisch bemessen oder können mit der {{cssxref("grid-auto-columns")}} Eigenschaft eine spezifische Größe erhalten.

Bei der Arbeit mit der Ausrichtung im [CSS Grid Layout](/de/docs/Web/CSS/CSS_grid_layout) wird die Achse, entlang derer die Spalten verlaufen, als _Block- oder Spaltenachse_ bezeichnet.
