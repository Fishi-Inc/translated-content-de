Eine **Schleife** ist eine Abfolge von Anweisungen, die wiederholt wird, bis eine bestimmte Bedingung erfüllt ist. Ein Beispiel wäre der Vorgang, ein Datenelement zu holen und es zu verändern, und dann sicherzustellen, dass eine bestimmte {{Glossary("conditional", "Bedingung")}} überprüft wird — zum Beispiel, ob ein Zähler eine vorgeschriebene Zahl erreicht hat.

Schleifen sind eine Möglichkeit, eine Anweisung für eine variable Anzahl von Zeiten auszuführen. Der gleiche Effekt kann mit {{glossary("Recursion", "Rekursion")}} erreicht werden, insbesondere in Sprachen, in denen alle Daten {{glossary("Immutable", "unveränderlich")}} sind und es daher unmöglich ist, eine Zählervariable zu aktualisieren.
