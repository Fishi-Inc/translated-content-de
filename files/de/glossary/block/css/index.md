Ein **Block** auf einer Webseite ist ein {{glossary("HTML")}}-{{glossary("element")}}, das in einem horizontalen Schreibmodus auf einer neuen Zeile erscheint, d.h. unter dem vorhergehenden Element und über dem folgenden Element (allgemein bekannt als _Block-Level-Element_). Zum Beispiel ist {{htmlelement("p")}} standardmäßig ein Block-Level-Element, während {{htmlelement("a")}} ein _inline-Element_ ist — Sie können mehrere Links nebeneinander in Ihrem HTML-Quellcode platzieren, und sie werden in der gerenderten Ausgabe in derselben Zeile erscheinen.

Mit der {{cssxref("display")}}-Eigenschaft können Sie ändern, ob ein Element inline oder als Block angezeigt wird (unter vielen anderen Optionen); **Blöcke** unterliegen auch den Effekten von Positionierungsschemata und der Verwendung der {{cssxref("position")}}-Eigenschaft.
