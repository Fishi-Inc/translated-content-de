In {{glossary("JavaScript")}}, ist ein **Block** eine Sammlung von verwandten {{glossary("statement","Anweisungen")}}, die in geschweifte Klammern ("{}") eingeschlossen sind. Zum Beispiel können Sie einen Block von Anweisungen nach einem {{jsxref("Statements/if...else","if (condition)")}} Block setzen, was anzeigt, dass der Interpreter den Code innerhalb des Blocks ausführen soll, wenn die Bedingung wahr ist, oder den gesamten Block überspringen soll, wenn die Bedingung falsch ist.
