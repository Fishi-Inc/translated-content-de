Ein **Identitätsanbieter** (IdP) ist eine Einheit in einem {{glossary("federated identity", "föderierten Identitäts")}}system, das die {{glossary("credential", "Anmeldedaten")}} eines Benutzers verwaltet und Benutzer {{glossary("authentication", "authentifizieren")}} kann.

In föderierten Identitätssystemen lagern {{glossary("relying party", "vertrauende Parteien")}}, die den Zugriff auf eine Ressource kontrollieren müssen (zum Beispiel eine Website, die entscheidet, ob ein Benutzer angemeldet werden soll), die Aufgabe der Authentifizierung von Benutzern an einen Dritten aus, dem sie vertrauen, um Authentifizierungsentscheidungen zu treffen. Diese Drittparteien werden Identitätsanbieter genannt.

Beispiele für Identitätsanbieter im Web sind Google, Microsoft und Facebook. Dies ermöglicht es Websites, Benutzern die Möglichkeit zu geben, sich mit dem Google-, Microsoft- oder Facebook-Konto des Nutzers anzumelden.
