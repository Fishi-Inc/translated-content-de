**Real User Monitoring** oder RUM misst die Leistung einer Seite von den Geräten echter Benutzer. In der Regel injiziert ein Drittanbieterskript ein Skript auf jeder Seite, um die Seitendaten für jede Anfrage zu messen und zu melden. Diese Technik überwacht die tatsächlichen Benutzerinteraktionen einer Anwendung. Im RUM sammelt das Drittanbieterskript Leistungskennzahlen von den Browsern der realen Benutzer. RUM hilft dabei zu identifizieren, wie eine Anwendung genutzt wird, einschließlich der geografischen Verteilung der Benutzer und deren Einfluss auf die Benutzererfahrung.
