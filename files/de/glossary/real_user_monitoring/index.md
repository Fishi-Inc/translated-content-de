**Real User Monitoring** oder RUM misst die Leistung einer Seite von den Maschinen echter Benutzer. In der Regel fügt ein Skript eines Drittanbieters ein Skript auf jeder Seite ein, um die Ladezeiten der Seite für jede Anfrage zu messen und zu melden. Diese Technik überwacht die tatsächlichen Benutzerinteraktionen einer Anwendung. Bei RUM sammelt das Skript eines Drittanbieters Leistungsmetriken aus den Browsern der echten Benutzer. RUM hilft dabei, zu erkennen, wie eine Anwendung genutzt wird, einschließlich der geografischen Verteilung der Benutzer und der Auswirkungen dieser Verteilung auf die Endbenutzererfahrung.
