Ein **MIME-Typ** (korrekt bezeichnet als "media type", aber auch manchmal "content type") ist eine Zeichenfolge, die zusammen mit einer Datei gesendet wird und den Typ der Datei angibt (beschreibt das Inhaltsformat, zum Beispiel könnte eine Audiodatei als `audio/ogg` gekennzeichnet sein oder eine Bilddatei als `image/png`).

Er erfüllt denselben Zweck wie Dateierweiterungen traditionell in Windows.
Der Name stammt vom [MIME](/de/docs/Glossary/mime)-Standard, der ursprünglich in E-Mails verwendet wurde.
