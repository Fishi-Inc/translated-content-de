{{GlossarySidebar}}

Die **Tabellenraster-Box** ist eine Block-Element-Box, die alle internen Tabellen-Boxen enthält, mit Ausnahme der Überschrift. Die Box, die die Überschrift einschließt, wird als {{Glossary("table wrapper box")}} bezeichnet.
