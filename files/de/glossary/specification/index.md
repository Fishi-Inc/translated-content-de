Eine **Spezifikation** ist ein Dokument, das im Detail festlegt, welche Funktionen oder Eigenschaften ein Produkt vor der Auslieferung enthalten muss. Im Kontext der Beschreibung des Webs bedeutet der Begriff "Spezifikation" (oft verkürzt zu "Spec") im Allgemeinen ein Dokument, das eine Sprache, Technologie oder ein [API](/de/docs/Glossary/API) beschreibt, die das vollständige Set offener Webtechnologien bilden.
