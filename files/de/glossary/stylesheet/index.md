Ein **Stylesheet** ist eine Sammlung von CSS-Regeln, die verwendet werden, um das Layout und Design einer Webseite oder eines Dokuments zu steuern. _Interne_ Stylesheets werden innerhalb eines {{htmlelement("style")}}-Elements im {{htmlelement("head")}} eines Webdokuments platziert, während _externe_ Stylesheets in einer separaten `.css`-Datei abgelegt werden, die auf ein Dokument angewendet wird, indem die Datei innerhalb eines {{htmlelement("link")}}-Elements im Kopf des Dokuments referenziert wird.

Externe Stylesheets werden allgemein bevorzugt, da sie Ihnen ermöglichen, das Styling mehrerer Seiten von einem einzigen Ort aus zu steuern, anstatt das CSS auf jeder Seite wiederholen zu müssen.
