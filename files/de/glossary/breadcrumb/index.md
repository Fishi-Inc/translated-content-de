Ein **Breadcrumb** oder Breadcrumb-Pfad ist eine Navigation, die typischerweise zwischen dem Header einer Seite und dem Hauptinhalt platziert wird. Sie zeigt entweder eine Hierarchie der aktuellen Seite in Bezug auf die Struktur der Website an, von der obersten Ebene zur aktuellen Seite, oder eine Liste der Links, denen der Benutzer gefolgt ist, um zur aktuellen Seite zu gelangen, in der Reihenfolge, in der sie besucht wurden.

Ein lokaler Breadcrumb für dieses Dokument könnte etwa so aussehen:

[MDN](/) > [Glossar](/de/docs/Glossary) > Breadcrumb

Breadcrumb-Pfade ermöglichen es Benutzern, sich ihrer Position innerhalb einer Website bewusst zu sein. Diese Art der Navigation hilft den Benutzern, wenn sie korrekt umgesetzt wird, zu wissen, wo sie sich auf einer Website befinden und wie sie dorthin gelangt sind. Sie können einem Benutzer auch helfen, zu einer vorherigen Position zurückzukehren und die Anzahl der Klicks zu reduzieren, die notwendig sind, um zu einer höherstufigen Seite zu gelangen.

> [!NOTE]
> Auch wenn es kein spezielles [semantisches HTML-Element](https://html.spec.whatwg.org/multipage/semantics-other.html#rel-up) für Breadcrumb-Navigationsmenüs gibt, wird das {{htmlelement("ol")}}-Element häufig verwendet, um ihre hierarchische Struktur darzustellen.
