Ein **Breadcrumb**, oder Brotkrümelpfad, ist eine Navigationshilfe, die typischerweise zwischen dem Header einer Website und dem Hauptinhalt platziert wird. Sie zeigt entweder eine Hierarchie der aktuellen Seite in Bezug auf die Struktur der Website, vom obersten Level bis zur aktuellen Seite, oder eine Liste der Links, denen der Benutzer gefolgt ist, um zur aktuellen Seite zu gelangen, in der Reihenfolge, in der sie besucht wurden.

Ein Standorthinweis für dieses Dokument könnte folgendermaßen aussehen:

[MDN](/) > [Glossary](/de/docs/Glossary) > Breadcrumb

Breadcrumb-Pfade ermöglichen es den Benutzern, sich ihrer Position innerhalb einer Website bewusst zu sein. Diese Art der Navigation, wenn sie korrekt umgesetzt wird, hilft den Benutzern zu verstehen, wo sie sich auf einer Seite befinden und wie sie dort hingelangt sind. Sie können einem Benutzer auch helfen, zu einer vorherigen Position zurückzukehren und die Anzahl der Klicks zu reduzieren, die notwendig sind, um zu einer Seite höherer Ebene zu gelangen.

> [!NOTE]
> Obwohl es kein spezielles [semantisches HTML-Element](https://html.spec.whatwg.org/multipage/semantics-other.html#rel-up) für Breadcrumb-Navigationsmenüs gibt, wird das {{htmlelement("ol")}}-Element häufig verwendet, um deren hierarchische Struktur darzustellen.
