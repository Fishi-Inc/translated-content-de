Ein **Cookie** ist ein kleines Informationsstück, das von einer Website über einen Webbrowser auf dem Computer eines Besuchers hinterlassen wird.

Cookies werden verwendet, um die Web-Erfahrung eines Benutzers mit einer Website zu personalisieren. Sie können die Präferenzen oder Eingaben eines Benutzers enthalten, wenn er auf diese Website zugreift. Ein Benutzer kann seinen Webbrowser so anpassen, dass er Cookies akzeptiert, ablehnt oder löscht.

Cookies können auf der Serverseite über den `Set-Cookie` [HTTP-Header](/de/docs/Web/HTTP/Cookies) gesetzt und modifiziert werden oder mit JavaScript unter Verwendung von [`document.cookie`](/de/docs/Web/API/Document/cookie).
