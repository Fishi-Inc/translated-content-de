Das _World Wide Web_—häufig als **WWW**, **W3** oder **das Web** bezeichnet—ist ein System miteinander verbundener öffentlicher Webseiten, auf die über das {{Glossary("Internet")}} zugegriffen werden kann. Das Web ist nicht dasselbe wie das Internet: Das Web ist eine von vielen Anwendungen, die auf dem Internet aufbauen.

Tim Berners-Lee schlug die Architektur dessen vor, was als World Wide Web bekannt wurde. Er erstellte den ersten Web-{{Glossary("Server","server")}}, Web-{{Glossary("Browser","browser")}} und die erste Webseite auf seinem Computer im CERN-Forschungszentrum für Physik im Jahr 1990. 1991 gab er seine Erfindung in der Newsgroup alt.hypertext bekannt und markierte damit den Moment, in dem das Web erstmals der Öffentlichkeit zugänglich gemacht wurde.

Das System, das wir heute als "das Web" kennen, besteht aus mehreren Komponenten:

- Das **{{Glossary("HTTP")}}**-Protokoll regelt die Datenübertragung zwischen einem Server und einem Client.
- Um auf eine Web-Komponente zuzugreifen, gibt ein Client einen eindeutigen universellen Bezeichner an, der als **{{Glossary("URL")}}** (Uniform Resource Locator) oder {{Glossary("URI")}} (Uniform Resource Identifier) bezeichnet wird (formal als Universal Document Identifier (UDI) bekannt).
- **{{Glossary("HTML")}}** (Hypertext Markup Language) ist das gebräuchlichste Format zur Veröffentlichung von Webdokumenten.

Das Verlinken oder Verbinden von Ressourcen durch {{Glossary("Hyperlink","hyperlinks")}} ist ein grundlegendes Konzept des Webs und unterstützt dessen Identität als Sammlung verbundener Dokumente.

Kurz nach der Erfindung des Webs gründete Tim Berners-Lee das {{Glossary("W3C")}} (World Wide Web Consortium), um das Web weiter zu standardisieren und zu entwickeln. Dieses Konsortium besteht aus zentralen Web-Interessengruppen wie Web-Browser-Entwicklern, Regierungsstellen, Forschern und Universitäten. Seine Mission umfasst Bildung und Öffentlichkeitsarbeit.
