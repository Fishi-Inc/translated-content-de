Die schreibgeschützte **`propertyName`**-Eigenschaft von {{domxref("TransitionEvent")}}-Objekten ist ein String, der den Namen der CSS-Eigenschaft enthält, die mit der Transition verbunden ist.

## Wert

Ein String.
