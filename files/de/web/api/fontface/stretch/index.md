Die **`stretch`**-Eigenschaft der [`FontFace`](/de/docs/Web/API/FontFace)-Schnittstelle ruft ab oder legt fest, wie die Schrift gedehnt wird.

Diese Eigenschaft entspricht dem {{cssxref("@font-face/font-stretch", "font-stretch")}}-Deskriptor.

## Wert

Ein String, der einen Deskriptor enthält, wie er in der `@font-face`-Regel eines Stylesheets definiert wäre.
