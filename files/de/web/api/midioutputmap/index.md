{{APIRef("Web MIDI API")}}{{SecureContext_Header}}

Die **`MIDIOutputMap`**-Schnittstelle des [Web MIDI API](/de/docs/Web/API/Web_MIDI_API) im Nur-Lese-Modus bietet die Menge der derzeit verfügbaren MIDI-Ausgabeports.

Eine Instanz von `MIDIOutputMap` ist ein Nur-Lese-[`Map`-ähnliches Objekt](/de/docs/Web/JavaScript/Reference/Global_Objects/Map#map-like_browser_apis), bei dem jeder Schlüssel die ID-Zeichenfolge für die MIDI-Ausgabe ist und der zugehörige Wert das entsprechende {{domxref("MIDIOutput")}}-Objekt ist.
