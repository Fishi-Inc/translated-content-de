Die **`Navigator.taintEnabled()`**-Methode gibt immer `false` zurück.

Tainting war eine Sicherheitstechnik, die in JavaScript 1.2 verwendet wurde. Sie wurde schon vor langer Zeit entfernt; diese Methode bleibt nur, um die Kompatibilität mit sehr alten Skripten zu gewährleisten.

## Syntax

```js-nolint
taintEnabled()
```

### Parameter

Keine.

### Rückgabewert

Gibt immer `false` zurück.
