{{APIRef("HTML DOM")}}

Die readonly-Eigenschaft **`form`** des {{domxref("HTMLInputElement")}}-Interfaces gibt ein {{domxref("HTMLFormElement")}}-Objekt zurück, das dieses {{HTMLElement("input")}} besitzt, oder `null`, wenn dieses Input-Element zu keinem Formular gehört.

## Wert

Ein {{domxref("HTMLFormElement")}} oder `null`.
