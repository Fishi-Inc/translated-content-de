Die **`close()`**-Methode der {{domxref("USBDevice")}}
Schnittstelle gibt ein {{jsxref("promise")}} zurück, das sich erfüllt, wenn alle offenen Schnittstellen freigegeben sind und die Gerätesitzung beendet wurde.

## Syntax

```js-nolint
close()
```

### Parameter

Keine.

### Rückgabewert

Ein {{jsxref("promise")}}.
