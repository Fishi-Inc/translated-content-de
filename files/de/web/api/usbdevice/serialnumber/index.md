Die **`serialNumber`**-Eigenschaft des `[USBDevice](/de/docs/Web/API/USBDevice)`-Interfaces ist die herstellerdefinierte Seriennummer für das spezifische USB-Gerät.

## Wert

Die Seriennummer für das angegebene USB-Gerät
