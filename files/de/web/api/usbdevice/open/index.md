Die **`open()`**-Methode des [`USBDevice`](/de/docs/Web/API/USBDevice)-Interface gibt ein {{jsxref("promise")}} zurück, das aufgelöst wird, wenn eine Gerätesitzung gestartet wurde.

## Syntax

```js-nolint
open()
```

### Parameter

Keine.

### Rückgabewert

Ein {{jsxref("promise")}}.
