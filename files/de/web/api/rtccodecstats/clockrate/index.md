{{APIRef("WebRTC")}}

Die **`clockRate`**-Eigenschaft des {{domxref("RTCCodecStats")}} Wörterbuchs ist eine positive Zahl, die die Medienabtastrate in Hertz (Hz) enthält.

## Werte

Eine positive Zahl, die die Medienabtastrate angibt.
