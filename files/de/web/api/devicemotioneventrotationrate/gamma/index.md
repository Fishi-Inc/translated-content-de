Die schreibgeschützte **`gamma`**-Eigenschaft des [`DeviceMotionEventRotationRate`](/de/docs/Web/API/DeviceMotionEventRotationRate)-Interfaces gibt die Rotationsgeschwindigkeit um die Y-Achse in Grad pro Sekunde an.

## Wert

Ein `double`, das die Rotationsgeschwindigkeit um die Y-Achse in Grad pro Sekunde angibt. Details finden Sie in den [Erläuterungen zu Beschleunigerwerten](/de/docs/Web/API/Device_orientation_events/Detecting_device_orientation#accelerometer_values_explained).
