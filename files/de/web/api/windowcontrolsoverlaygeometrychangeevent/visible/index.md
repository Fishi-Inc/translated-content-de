Die **`visible`** schreibgeschützte Eigenschaft der {{domxref("WindowControlsOverlayGeometryChangeEvent")}} Schnittstelle ist ein boolesches Kennzeichen, das angibt, ob die Fenstersteuerungsüberlagerung in einer desktop-installierten Progressive Web App sichtbar ist oder nicht.

Wenn der Benutzer die [Window Controls Overlay](/de/docs/Web/API/Window_Controls_Overlay_API) Funktion deaktiviert, erscheint die standardmäßige Titelleiste und die Fenstersteuerungstasten erscheinen nicht als Überlagerung. In diesem Fall ist das `visible` Kennzeichen false.
