[`DocumentFragment`](/de/docs/Web/API/DocumentFragment)-Objekt zurück.

## Syntax

```js-nolint
new DocumentFragment()
```

### Parameter

Keine.

### Rückgabewert

Ein neues [`DocumentFragment`](/de/docs/Web/API/DocumentFragment)-Objekt.
