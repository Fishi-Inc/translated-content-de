Die **`city`**-Eigenschaft des {{domxref('PaymentAddress')}}-Interfaces gibt einen String zurück, der den Stadt- oder Ortsabschnitt der Adresse enthält.

## Wert

Ein String, der den Stadt- oder Ortsabschnitt der in dem {{domxref("PaymentAddress")}}-Objekt beschriebenen Adresse angibt.
