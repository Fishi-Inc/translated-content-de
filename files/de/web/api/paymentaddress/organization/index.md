Die **`organization`** schreibgeschützte Eigenschaft der {{domxref('PaymentAddress')}}-Schnittstelle gibt einen String zurück, der den Namen der Organisation, Firma, Unternehmen oder Institution an der Adresse enthält.

## Wert

Ein String, dessen Wert der Name der Organisation oder des Unternehmens ist, die an der durch das `PaymentAddress`-Objekt beschriebenen Adresse angesiedelt ist. Dies sollte der Name der Organisation sein, die die Lieferung an Versandadressen erhalten soll, oder die für Zahlungsadressen verantwortlich ist.
