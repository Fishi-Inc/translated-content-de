Die **`organization`** schreibgeschützte
Eigenschaft des [`PaymentAddress`](/de/docs/Web/API/PaymentAddress)-Interfaces gibt einen String zurück, der den Namen der Organisation, Firma, Unternehmen oder Institution an der Adresse enthält.

## Wert

Ein String, dessen Wert der Name der Organisation oder des Unternehmens ist,
das sich an der Adresse befindet, die durch das `PaymentAddress`-Objekt beschrieben wird. Dies sollte der Name der Organisation sein, die für Lieferadressen die Sendung erhalten soll, oder die für Zahlungsadressen verantwortlich ist.
