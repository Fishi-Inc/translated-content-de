Die **`BluetoothRemoteGATTCharacteristic.stopNotifications()`**-Methode
gibt ein {{jsxref("Promise")}} an die BluetoothRemoteGATTCharacteristic-Instanz zurück, wenn keine aktive Benachrichtigung mehr vorliegt.

## Syntax

```js-nolint
stopNotifications()
```

### Parameter

Keine.

### Rückgabewert

Ein {{jsxref("Promise")}}.
