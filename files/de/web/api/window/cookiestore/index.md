Die schreibgeschützte **`cookieStore`**-Eigenschaft der [`Window`](/de/docs/Web/API/Window)-Schnittstelle gibt eine Referenz auf das [`CookieStore`](/de/docs/Web/API/CookieStore)-Objekt für den aktuellen Dokumentkontext zurück. Dies ist ein Einstiegspunkt für die [Cookie Store API](/de/docs/Web/API/Cookie_Store_API).

## Wert

Eine [`CookieStore`](/de/docs/Web/API/CookieStore)-Objektinstanz.
