{{APIRef}}

**`window.document`** gibt eine Referenz auf das im Fenster enthaltene [Dokument](/de/docs/Web/API/Document) zurück.

## Wert

Ein [Document](/de/docs/Web/API/Document) Objekt.

## Beispiele

```js
console.log(window.document.title);
```
