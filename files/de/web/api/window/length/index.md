{{APIRef}}

Gibt die Anzahl der Frames (entweder {{HTMLElement("frame")}} oder
{{HTMLElement("iframe")}} Elemente) im Fenster zurück.

## Wert

Eine Zahl.

## Beispiele

```js
if (window.length) {
  // this is a document with subframes
}
```
