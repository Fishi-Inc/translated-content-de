{{APIRef("HTML DOM")}}

Die **`target`**-Eigenschaft der {{domxref("HTMLFormElement")}}-Schnittstelle gibt das Ziel der Aktion des Formulars an (d.h. das Frame, in dem die Ausgabe gerendert werden soll).

## Wert

Ein Zeichenkette.

## Beispiele

```js
myForm.target = document.frames[1].name;
```
