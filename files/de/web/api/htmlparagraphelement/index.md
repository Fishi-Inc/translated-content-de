{{ APIRef("HTML DOM") }}

Die **`HTMLParagraphElement`**-Schnittstelle bietet spezielle Eigenschaften (über die des regulären [`HTMLElement`](/de/docs/Web/API/HTMLElement)-Objekts hinaus, das sie erbt) zur Manipulation von {{HTMLElement("p")}}-Elementen.

{{InheritanceDiagram}}

## Instanz-Eigenschaften

_Erbt Eigenschaften von ihrer übergeordneten Schnittstelle, [`HTMLElement`](/de/docs/Web/API/HTMLElement)._

- [`HTMLParagraphElement.align`](/de/docs/Web/API/HTMLParagraphElement/align) {{deprecated_inline}}
  - : Ein String, der eine aufgezählte Eigenschaft darstellt, die die Ausrichtung des Inhalts des Elements im Hinblick auf den umgebenden Kontext angibt. Die möglichen Werte sind `"left"`, `"right"`, `"justify"` und `"center"`.

## Instanz-Methoden

_Keine spezifischen Methoden, erbt Methoden von ihrer übergeordneten Schnittstelle, [`HTMLElement`](/de/docs/Web/API/HTMLElement)._
