{{APIRef("HTML DOM")}}

Die **`HTMLMediaElement.muted`**-Eigenschaft gibt an, ob das Media-Element stummgeschaltet ist.

## Wert

Ein booleanischer Wert. `true` bedeutet stummgeschaltet und `false` bedeutet
nicht stummgeschaltet.

## Beispiele

```js
const obj = document.createElement("video");
console.log(obj.muted); // false
```
