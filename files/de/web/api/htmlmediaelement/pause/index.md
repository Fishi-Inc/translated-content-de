{{APIRef("HTML DOM")}}

Die **`HTMLMediaElement.pause()`**-Methode pausiert die Wiedergabe
des Mediums. Befindet sich das Medium bereits im pausierten Zustand, hat diese Methode keine Auswirkung.

## Syntax

```js-nolint
pause()
```

### Parameter

Keine.

### Rückgabewert

Keiner ({{jsxref("undefined")}}).

### Ausnahmen

Keine.
