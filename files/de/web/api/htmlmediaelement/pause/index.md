{{APIRef("HTML DOM")}}

Die **`HTMLMediaElement.pause()`**-Methode pausiert die Wiedergabe der Medien. Wenn die Medien bereits pausiert sind, hat diese Methode keine Wirkung.

## Syntax

```js-nolint
pause()
```

### Parameter

Keine.

### Rückgabewert

Keine ({{jsxref("undefined")}}).

### Ausnahmen

Keine.
