Die **`size`**-Eigenschaft der {{domxref("CustomStateSet")}}-Schnittstelle gibt die Anzahl der Werte im `CustomStateSet` zurück.

## Wert

Eine Ganzzahl, die angibt, wie viele Eigenschaften das `CustomStateSet` hat.
