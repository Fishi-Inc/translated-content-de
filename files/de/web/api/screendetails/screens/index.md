Die schreibgeschützte **`screens`**-Eigenschaft der {{domxref("ScreenDetails")}}-Schnittstelle enthält ein Array von {{domxref("ScreenDetailed")}}-Objekten, die jeweils detaillierte Informationen über einen spezifischen Bildschirm des Geräts des Benutzers repräsentieren.

## Wert

Ein Array von {{domxref("ScreenDetailed")}}-Objekten.

> **Note:** `screens` umfasst nur "erweiterte" Anzeigen, nicht solche, die eine andere Anzeige spiegeln.

## Beispiele

Sehen Sie sich die Hauptseite [`ScreenDetails`](/de/docs/Web/API/ScreenDetails#examples) für Anwendungsbeispiele an.
