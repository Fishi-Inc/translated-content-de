Die schreibgeschützte Eigenschaft **`activated`** des {{domxref("Sensor")}}-Interfaces gibt einen booleschen Wert zurück, der anzeigt, ob der Sensor aktiv ist.

Da {{domxref('Sensor')}} eine Basisklasse ist, kann `activated` nur von einer ihrer abgeleiteten Klassen ausgelesen werden.

## Wert

Ein boolescher Wert.
