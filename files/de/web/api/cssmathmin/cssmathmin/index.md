Der **`CSSMathMin()`**-Konstruktor erstellt ein
neues {{domxref("CSSMathMin")}}-Objekt, das die CSS
{{CSSXref('min','min()')}}-Funktion repräsentiert.

## Syntax

```js-nolint
new CSSMathMin(args)
```

### Parameter

- `args`
  - : Eine Liste von Werten für das {{domxref('CSSMathProduct')}}-Objekt, die entweder ein Gleitkommazahl-Integer oder ein {{domxref('CSSNumericValue')}} sein können.

### Ausnahmen

- [`TypeError`](/de/docs/Web/JavaScript/Reference/Global_Objects/TypeError)
  - : Wird ausgelöst, wenn ein _Fehler_ beim Hinzufügen aller Werte in args auftritt.
