{{APIRef("HTML DOM")}}{{deprecated_header}}

Die **`ch`**-Eigenschaft der Schnittstelle [`HTMLTableCellElement`](/de/docs/Web/API/HTMLTableCellElement) hat keine Funktion. Sie spiegelt das `char`-Attribut des Zellelements wider.

> [!NOTE]
> Diese Eigenschaft wurde entwickelt, um die Möglichkeit zu bieten, Tabellenzelleninhalte an einem bestimmten Zeichen (typischerweise dem Dezimalpunkt) auszurichten, wurde jedoch nie von Browsern implementiert.
>
> Um eine solche Ausrichtung zu erreichen, achten Sie auf die Unterstützung eines Zeichenfolgenwerts mit der CSS-Eigenschaft {{cssxref("text-align")}}.

## Wert

Ein einzelnes Zeichen.
