{{APIRef("HTML DOM")}}{{deprecated_header}}

Die **`ch`** Eigenschaft des {{domxref("HTMLTableCellElement")}}-Interfaces tut nichts. Sie spiegelt das `char`-Attribut des Zellelements wider.

> [!NOTE]
> Diese Eigenschaft war dazu gedacht, die Möglichkeit zu bieten, Tabellenzelleninhalt an einem bestimmten Zeichen (typischerweise dem Dezimalpunkt) auszurichten, wurde jedoch nie von Browsern implementiert.
>
> Um eine solche Ausrichtung zu erreichen, achten Sie auf die Unterstützung eines Zeichenfolgenwerts mit der {{cssxref("text-align")}} CSS-Eigenschaft.

## Wert

Ein einzelnes Zeichen.
