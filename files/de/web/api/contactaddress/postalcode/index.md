Die schreibgeschützte **`postalCode`**-Eigenschaft des {{domxref("ContactAddress")}}-Interfaces gibt einen String zurück, der einen von einer Behörde für die Postzustellung verwendeten Code enthält, zum Beispiel die [PLZ](https://en.wikipedia.org/wiki/ZIP_Code) in den Vereinigten Staaten oder die [Postleitzahl](https://en.wikipedia.org/wiki/Postal_Index_Number) (PIN-Code) in Indien.

## Wert

Ein String, der den Postleitzahlenteil der Adresse enthält. Eine Postleitzahl ist ein String (entweder numerisch oder alphanumerisch), der von einem Postdienst verwendet wird, um die Postweiterleitung und -zustellung zu optimieren.

In verschiedenen Ländern werden unterschiedliche Begriffe dafür verwendet. In den meisten Teilen der Welt ist es als "Post code" oder "postal code" bekannt. In den Vereinigten Staaten wird der Begriff ZIP code verwendet. Indien verwendet PIN-Codes.
