## Instanz-Eigenschaften

_Diese Schnittstelle erbt auch Eigenschaften von ihrer übergeordneten Schnittstelle, {{domxref("SVGElement")}}._

- {{domxref("SVGFEDiffuseLightingElement.diffuseConstant")}} {{ReadOnlyInline}}
  - : Ein {{domxref("SVGAnimatedNumber")}}, der dem {{SVGAttr("diffuseConstant")}} Attribut des gegebenen Elements entspricht.
- {{domxref("SVGFEDiffuseLightingElement.height")}} {{ReadOnlyInline}}
  - : Ein {{domxref("SVGAnimatedLength")}}, der dem {{SVGAttr("height")}} Attribut des gegebenen Elements entspricht.

  - : Ein {{domxref("SVGAnimatedString")}}, der dem {{SVGAttr("in")}} Attribut des gegebenen Elements entspricht.
- {{domxref("SVGFEDiffuseLightingElement.kernelUnitLengthX")}} {{ReadOnlyInline}}
  - : Ein {{domxref("SVGAnimatedNumber")}}, der der X-Komponente des {{SVGAttr("kernelUnitLength")}} Attributs des gegebenen Elements entspricht.
- {{domxref("SVGFEDiffuseLightingElement.kernelUnitLengthY")}} {{ReadOnlyInline}}
  - : Ein {{domxref("SVGAnimatedNumber")}}, der der Y-Komponente des {{SVGAttr("kernelUnitLength")}} Attributs des gegebenen Elements entspricht.

- {{domxref("SVGFEDiffuseLightingElement.width")}} {{ReadOnlyInline}}
  - : Ein {{domxref("SVGAnimatedLength")}}, der dem {{SVGAttr("width")}} Attribut des gegebenen Elements entspricht.
- {{domxref("SVGFEDiffuseLightingElement.x")}} {{ReadOnlyInline}}
  - : Ein {{domxref("SVGAnimatedLength")}}, der dem {{SVGAttr("x")}} Attribut des gegebenen Elements entspricht.
- {{domxref("SVGFEDiffuseLightingElement.y")}} {{ReadOnlyInline}}
  - : Ein {{domxref("SVGAnimatedLength")}}, der dem {{SVGAttr("y")}} Attribut des gegebenen Elements entspricht.
