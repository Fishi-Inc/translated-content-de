{{APIRef}}

Der **`Document`**-Konstruktor erstellt ein neues
{{domxref("Document")}}-Objekt, das eine im Browser geladene Webseite darstellt und als
Einstiegspunkt in die Inhalte der Seite dient.

## Syntax

```js-nolint
new Document()
```

### Parameter

Keine.
