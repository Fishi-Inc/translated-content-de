Der **`Document`** Konstruktor erstellt ein neues
[`Document`](/de/docs/Web/API/Document) Objekt, das eine im Browser geladene Webseite ist und als Einstiegspunkt in den Inhalt der Seite dient.

## Syntax

```js-nolint
new Document()
```

### Parameter

Keine.
