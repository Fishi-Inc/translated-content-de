> [!WARNING]
> Diese Schnittstelle dieser Eigenschaft ist im [Navigation Timing Level 2-Spezifikation](https://w3c.github.io/navigation-timing/#obsolete) veraltet. Bitte verwenden Sie stattdessen die [`PerformanceNavigationTiming`](/de/docs/Web/API/PerformanceNavigationTiming)
> Schnittstelle.

Die veraltete
**`PerformanceTiming.domContentLoadedEventEnd`**
read-only Eigenschaft gibt einen `unsigned long long` zurück, der den Moment darstellt,
in Millisekunden seit dem UNIX-Epoch, unmittelbar nachdem alle Skripte, die so schnell wie möglich
ausgeführt werden müssen, in beliebiger Reihenfolge, ausgeführt wurden.

## Wert

Ein `unsigned long long`.
