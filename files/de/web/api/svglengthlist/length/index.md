Die **`length`**-Eigenschaft des [`SVGLengthList`](/de/docs/Web/API/SVGLengthList)-Interfaces gibt die Anzahl der Elemente in der Liste zurück. Sie ist ein Alias von [`numberOfItems`](/de/docs/Web/API/SVGLengthList/numberOfItems), um SVG-Listen mehr [array-ähnlich](/de/docs/Web/JavaScript/Reference/Global_Objects/Array#array-like_objects) zu machen.

## Wert

Ein nicht-negativer Integer, der die Anzahl der Elemente in der Liste darstellt.

## Beispiele

Siehe [`SVGLengthList`](/de/docs/Web/API/SVGLengthList) für ein vollständiges Beispiel.
