Die **`clear()`** Methode der [`MIDIOutput`](/de/docs/Web/API/MIDIOutput)-Schnittstelle löscht die Warteschlange von Nachrichten, die an das Ausgabegerät gesendet werden.

## Syntax

```js-nolint
clear()
```

### Parameter

Keine.

### Rückgabewert

Keiner ({{jsxref("undefined")}}).
