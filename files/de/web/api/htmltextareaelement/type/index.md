{{ApiRef("HTML DOM")}}

Die schreibgeschützte **`type`**-Eigenschaft der {{domxref("HTMLTextAreaElement")}}-Schnittstelle gibt den String `"textarea"` zurück.

## Wert

Der String `"textarea"`.

## Beispiel

```js
const textArea = document.querySelector("textarea");
console.log(textArea.type); // "textarea"
```
