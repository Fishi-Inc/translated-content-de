Die **`CSSTransformValue.values()`** -Methode gibt ein
neues _Array-Iterator_-Objekt zurück, das die Werte für
jeden Index im CSSTransformValue-Objekt enthält.

## Syntax

```js-nolint
values()
```

### Parameter

Keine.

### Rückgabewert

Ein neues {{jsxref("Array")}}.
