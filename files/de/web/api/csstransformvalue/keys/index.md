Die **`CSSTransformValue.keys()`**-Methode
gibt ein neues _Array-Iterator_-Objekt zurück, das die Schlüssel
für jeden Index im Array enthält.

## Syntax

```js-nolint
keys()
```

### Parameter

Keine.

### Rückgabewert

Ein neues {{jsxref("Array")}}.
