{{APIRef("HTML DOM")}}

Die schreibgeschützte **`viewTransition`**-Eigenschaft des [`PageRevealEvent`](/de/docs/Web/API/PageRevealEvent)-Interfaces enthält ein [`ViewTransition`](/de/docs/Web/API/ViewTransition)-Objekt, das die aktive View-Transition für die dokumentübergreifende Navigation darstellt.

## Wert

Ein [`ViewTransition`](/de/docs/Web/API/ViewTransition)-Objekt oder `null`, wenn keine View-Transition aktiv ist, wenn das Ereignis ausgelöst wird.

## Beispiele

Siehe die Hauptseite [`PageRevealEvent`](/de/docs/Web/API/PageRevealEvent).
