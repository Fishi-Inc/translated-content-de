{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

Die **`clear()`** Methode des [`FontFaceSet`](/de/docs/Web/API/FontFaceSet)-Interfaces entfernt alle über dieses Interface hinzugefügten Schriftarten. Schriftarten, die mit der {{cssxref("@font-face")}}-Regel hinzugefügt wurden, werden nicht entfernt.

## Syntax

```js-nolint
clear()
```

### Parameter

Keine.

### Rückgabewert

Keiner ({{jsxref("undefined")}}).
