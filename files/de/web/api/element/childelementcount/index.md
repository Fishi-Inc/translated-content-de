{{ APIRef("DOM") }}

Die **`Element.childElementCount`** schreibgeschützte Eigenschaft gibt die Anzahl der Kindelemente dieses Elements zurück.

## Beispiele

```js
let sidebar = document.getElementById("sidebar");
if (sidebar.childElementCount > 0) {
  // Do something
}
```
