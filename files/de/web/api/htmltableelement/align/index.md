Die Eigenschaft **`HTMLTableElement.align`** gibt die Ausrichtung der Tabelle an.

## Wert

Einer der folgenden Zeichenfolgenwerte:

- `left`
- `center`
- `right`

## Beispiele

```js
// Setzen Sie die Ausrichtung einer Tabelle
const t = document.getElementById("TableA");
t.align = "center";
```
