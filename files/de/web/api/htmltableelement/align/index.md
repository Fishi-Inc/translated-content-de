Die **`HTMLTableElement.align`**-Eigenschaft steht für die
Ausrichtung der Tabelle.

## Wert

Einer der folgenden Zeichenkettenwerte:

- `left`
- `center`
- `right`

## Beispiele

```js
// Set the alignment of a table
const t = document.getElementById("TableA");
t.align = "center";
```
