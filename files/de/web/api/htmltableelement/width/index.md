Die **`HTMLTableElement.width`**-Eigenschaft repräsentiert die gewünschte Breite der Tabelle.

## Wert

Ein String, der die Breite in Pixeln oder als Prozentwert angibt.

## Beispiele

```js
mytable.width = "75%";
```
