{{APIRef("HTML DOM")}}

Die **`HTMLTableElement.caption`**-Eigenschaft repräsentiert die Tabellenbeschriftung. Falls kein Beschriftungselement der Tabelle zugeordnet ist, ist diese Eigenschaft `null`.

## Wert

Ein String.

## Beispiele

```js
if (table.caption) {
  // Do something with the caption
}
```
