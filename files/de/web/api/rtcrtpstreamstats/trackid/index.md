{{APIRef("WebRTC")}}

Die **`trackId`**-Eigenschaft des {{domxref("RTCRtpStreamStats")}} Wörterbuchs ist ein String, der das {{domxref("RTCMediaStreamTrackStats")}}-Objekt eindeutig identifiziert, welches die Track-Statistiken für den {{domxref("MediaStreamTrack")}} enthält, für den in diesem Objekt Statistiken bereitgestellt werden.

## Wert

Ein String, der das {{domxref("RTCMediaStreamTrackStats")}}-Objekt eindeutig identifiziert, das Statistiken für den Track bereitstellt, für den durch diesen {{domxref("RTCStatsReport")}} Statistiken gesammelt werden.

> [!NOTE]
> Dieser Wert ist _nicht_ derselbe wie der Wert von
> {{domxref("MediaStreamTrack.id")}}.
