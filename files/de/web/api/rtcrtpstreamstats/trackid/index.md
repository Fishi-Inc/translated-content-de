Die **`trackId`**-Eigenschaft des [`RTCRtpStreamStats`](/de/docs/Web/API/RTCRtpStreamStats)-Wörterbuchs ist ein String, der das [`RTCMediaStreamTrackStats`](/de/docs/Web/API/RTCMediaStreamTrackStats)-Objekt eindeutig identifiziert, das die Track-Statistiken für das [`MediaStreamTrack`](/de/docs/Web/API/MediaStreamTrack) bereitstellt, für das in diesem Objekt Statistiken bereitgestellt werden.

## Wert

Ein String, der das [`RTCMediaStreamTrackStats`](/de/docs/Web/API/RTCMediaStreamTrackStats)-Objekt eindeutig identifiziert, das Statistiken für die Spur bereitstellt, für die in diesem [`RTCStatsReport`](/de/docs/Web/API/RTCStatsReport) Statistiken gesammelt werden.

> [!NOTE]
> Dieser Wert ist _nicht_ identisch mit dem Wert von
> [`MediaStreamTrack.id`](/de/docs/Web/API/MediaStreamTrack/id).
