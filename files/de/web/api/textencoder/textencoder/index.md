{{APIRef("Encoding API")}}{{AvailableInWorkers}}

Der **`TextEncoder()`** Konstruktor gibt ein neu erstelltes [`TextEncoder`](/de/docs/Web/API/TextEncoder) Objekt zurück, das einen Byte-Stream mit UTF-8-Codierung erzeugt.

## Syntax

```js-nolint
new TextEncoder()
```

### Parameter

Keine.
