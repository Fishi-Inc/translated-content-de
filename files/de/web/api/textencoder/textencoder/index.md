{{APIRef("Encoding API")}}

Der **`TextEncoder()`** Konstruktor gibt ein neu erstelltes {{DOMxRef("TextEncoder")}} Objekt zurück, das einen Bytestrom mit UTF-8-Kodierung erzeugt.

## Syntax

```js-nolint
new TextEncoder()
```

### Parameter

Keine.
