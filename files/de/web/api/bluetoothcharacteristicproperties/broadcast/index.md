{{domxref("BluetoothCharacteristicProperties")}}-Schnittstelle gibt einen
`boolean` zurück, der `true` ist, wenn der Broadcast des Merkmalwerts mit dem Server Characteristic Configuration Descriptor erlaubt ist.

Ein boolescher Wert.
