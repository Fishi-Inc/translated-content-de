Die **`BluetoothRemoteGATTServer.disconnect()`**-Methode bewirkt,
dass die Skriptausführungsumgebung die Verbindung zu `this.device` trennt.

## Syntax

```js-nolint
disconnect()
```

### Parameter

Keine.

### Rückgabewert

Keiner ({{jsxref("undefined")}}).
