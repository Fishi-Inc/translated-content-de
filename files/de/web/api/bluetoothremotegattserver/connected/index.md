Die schreibgeschützte Eigenschaft **`BluetoothRemoteGATTServer.connected`**
gibt einen booleschen Wert zurück, der `true` ist, solange diese Skriptausführungsumgebung mit `this.device` verbunden ist. Es kann `false` sein, während der Benutzeragent physisch verbunden ist.

## Wert

Ein `boolean`.
