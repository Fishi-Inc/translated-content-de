Die **`BluetoothRemoteGATTServer.connect()`**-Methode veranlasst die Skriptausführungsumgebung, sich mit `this.device` zu verbinden.

## Syntax

```js-nolint
connect()
```

### Parameter

Keine.

### Rückgabewert

Ein {{jsxref("Promise")}}, der sich zu einem {{domxref("BluetoothRemoteGATTServer")}} auflöst.
