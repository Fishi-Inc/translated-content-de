{{APIRef("Encrypted Media Extensions")}}

Die **`values()`**-Methode der
{{domxref("MediaKeyStatusMap")}}-Schnittstelle gibt ein neues Iterator-Objekt zurück, das Werte für jedes Element in der Statuskarte in Einfüge-Reihenfolge enthält.

## Syntax

```js-nolint
values()
```

### Parameter

Keine.

### Rückgabewert

Ein neuer Iterator.
