{{domxref("CSSRotate")}}-Interfaces ruft den Drehwinkel ab und legt ihn fest. Ein positiver Winkel
bezeichnet eine Drehung im Uhrzeigersinn, ein negativer Winkel eine Drehung gegen den Uhrzeigersinn.
