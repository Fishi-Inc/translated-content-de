{{APIRef("CSS Typed Object Model API")}}

Die **`CSSRotate`**-Schnittstelle der [CSS Typed Object Model API](/de/docs/Web/API/CSS_Object_Model) repräsentiert den Rotationswert der einzelnen {{CSSXRef('transform')}}-Eigenschaft in CSS. Sie erbt Eigenschaften und Methoden von ihrer übergeordneten Schnittstelle {{domxref('CSSTransformValue')}}.

{{InheritanceDiagram}}

## Konstruktor

- {{domxref("CSSRotate.CSSRotate", "CSSRotate()")}}
  - : Erstellt ein neues `CSSRotate`-Objekt.

## Instanz-Eigenschaften

- {{domxref('CSSRotate.x','x')}}
  - : Gibt den Wert der x-Achse zurück oder setzt ihn.
- {{domxref('CSSRotate.y','y')}}
  - : Gibt den Wert der y-Achse zurück oder setzt ihn.
- {{domxref('CSSRotate.z','z')}}
  - : Gibt den Wert der z-Achse zurück oder setzt ihn.
- {{domxref('CSSRotate.angle','angle')}}
  - : Gibt den Winkelwert zurück oder setzt ihn.
