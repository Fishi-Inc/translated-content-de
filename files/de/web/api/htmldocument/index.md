{{APIRef("HTML DOM")}}

Aus historischen Gründen haben {{domxref("Window")}} Objekte eine `window.HTMLDocument` Eigenschaft, deren Wert das {{DOMxRef("Document")}} Interface ist. Sie können `HTMLDocument` also als Alias für {{DOMxRef("Document")}} betrachten, und Sie finden Dokumentation zu `HTMLDocument` Mitgliedern unter der Dokumentation für das {{DOMxRef("Document")}} Interface.
