{{APIRef("CSS Typed Object Model API")}}

Der **`CSSMathNegate()`**-Konstruktor erstellt ein neues {{domxref("CSSMathNegate")}}-Objekt, das den übergebenen Wert negiert.

## Syntax

```js-nolint
new CSSMathNegate(arg)
```

### Parameter

- `arg`
  - : Ein {{domxref('CSSNumericValue')}}.
