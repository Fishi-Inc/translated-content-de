Die **`encoding`**-Eigenschaft der {{DOMxRef("NDEFRecord")}}-Schnittstelle gibt die Kodierung einer textuellen Nutzlast zurück oder `null`, wenn keine vorhanden ist.

## Syntax

```js-nolint
NDEFRecord.encoding
```

### Wert

Ein String, der einer der folgenden Werte sein kann: `"utf-8"`, `"utf-16"`, `"utf-16le"` oder `"utf-16be"`.
