{{APIRef("Intersection Observer API")}}

Die Methode **`disconnect()`** des {{domxref("IntersectionObserver")}}-Objekts stoppt die Überwachung aller seiner Ziel-Elemente auf Sichtbarkeitsänderungen.

## Syntax

```js-nolint
disconnect()
```

### Parameter

Keine.

### Rückgabewert

Keiner ({{jsxref("undefined")}}).
