Der **`CSSSkewY()`**-Konstruktor erstellt ein neues [`CSSSkewY`](/de/docs/Web/API/CSSSkewY)-Objekt, das den Wert der [`skewY()`](/de/docs/Web/CSS/transform-function/skewY)-Funktion der individuellen {{CSSXRef('transform')}}-Eigenschaft in CSS darstellt.

## Syntax

```js-nolint
new CSSSkewY(ay)
```

### Parameter

- [`ay`](/de/docs/Web/API/CSSSkewY/ay)
  - : Ein Wert für den `ay`-Winkel des zu konstruierenden [`CSSSkewY`](/de/docs/Web/API/CSSSkewY)-Objekts. Dieser muss ein [`CSSNumericValue`](/de/docs/Web/API/CSSNumericValue) sein.
