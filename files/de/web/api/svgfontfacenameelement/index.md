{{APIRef("SVG")}}{{deprecated_header}}

Die **`SVGFontFaceNameElement`** Schnittstelle entspricht den {{SVGElement("font-face-name")}} Elementen.

Ein objektorientierter Zugriff auf die Attribute des {{SVGElement("font-face-name")}} Elements über das SVG DOM ist nicht möglich.

{{InheritanceDiagram}}

## Instanz-Eigenschaften

_Diese Schnittstelle hat keine eigenen Eigenschaften, erbt jedoch Eigenschaften von ihrem Elternteil, {{domxref("SVGElement")}}._

## Instanz-Methoden

_Diese Schnittstelle hat keine eigenen Methoden, erbt jedoch Methoden von ihrem Elternteil, {{domxref("SVGElement")}}._
