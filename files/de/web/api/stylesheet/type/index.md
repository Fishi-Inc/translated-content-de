{{APIRef("CSSOM")}}

Die **`type`**-Eigenschaft der [`StyleSheet`](/de/docs/Web/API/StyleSheet)-Schnittstelle gibt die Stylesheet-Sprache für das gegebene Stylesheet an.

## Wert

Ein String.

## Beispiele

```js
myStyleSheet.type = "text/css";
```
