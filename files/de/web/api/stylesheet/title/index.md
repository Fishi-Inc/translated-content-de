Die **`title`**-Eigenschaft der {{domxref("StyleSheet")}}-Schnittstelle gibt den empfohlenen Titel des aktuellen Stylesheets zurück.

Der Titel wird oft im {{domxref("StyleSheet/OwnerNode", "ownerNode")}} angegeben.
