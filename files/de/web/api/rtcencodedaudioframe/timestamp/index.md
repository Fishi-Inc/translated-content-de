Die schreibgeschützte **`timestamp`**-Eigenschaft des {{domxref("RTCEncodedAudioFrame")}}-Interfaces gibt die Zeit an, zu der die Rahmenabtastung begonnen hat.

## Wert

Eine positive ganze Zahl, die das Abtastzeitpunkt des ersten Bytes in diesem Rahmen in Mikrosekunden enthält.
