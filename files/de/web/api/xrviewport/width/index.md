Die schreibgeschützte [`XRViewport`](/de/docs/Web/API/XRViewport)-Eigenschaft **`width`** gibt die Breite des Viewports in Pixel an, auf die Zeichenfläche, auf die die 3D-Szene gerendert werden soll. Diese Breite wird zusammen mit der Höhe des Viewports, beschrieben durch dessen [`height`](/de/docs/Web/API/XRViewport/height)-Eigenschaft, und dem Ursprung, der durch die Eigenschaften [`x`](/de/docs/Web/API/XRViewport/x) und [`y`](/de/docs/Web/API/XRViewport/y) bestimmt ist, definiert.

## Wert

Die Breite des Viewports in Pixel.
