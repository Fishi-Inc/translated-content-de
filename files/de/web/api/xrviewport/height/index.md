Die schreibgeschützte [`XRViewport`](/de/docs/Web/API/XRViewport) Eigenschaft **`height`** gibt die Höhe in Pixeln des Ansichtsfensters auf der Zeichenoberfläche an, innerhalb derer die WebXR-Ansicht gerendert werden soll. Zusammen mit [`width`](/de/docs/Web/API/XRViewport/width) und dem durch [`x`](/de/docs/Web/API/XRViewport/x) und [`y`](/de/docs/Web/API/XRViewport/y) angegebenen Ursprungspunkt, definiert dies den Bereich, in dem gerenderte Inhalte gezeichnet werden.

## Wert

Die Höhe des Ansichtsfensters in Pixeln.
