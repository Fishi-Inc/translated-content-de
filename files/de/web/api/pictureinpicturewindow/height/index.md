{{APIRef("Picture-in-Picture API")}}

Die schreibgeschützte **`height`**-Eigenschaft des [`PictureInPictureWindow`](/de/docs/Web/API/PictureInPictureWindow) Interfaces gibt die Höhe des schwebenden Videofensters in Pixel zurück.

## Syntax

```js-nolint
pictureInPictureWindow.height
```

### Wert

Ein Integer-Wert, der die Höhe des schwebenden Videofensters in Pixeln angibt, falls das Picture-in-Picture-Fenster geöffnet ist. Andernfalls wird `0` zurückgegeben.
