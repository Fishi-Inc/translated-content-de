{{APIRef("HTML DOM")}}{{deprecated_header}}

Die **`ch`**-Eigenschaft der {{domxref("HTMLTableSectionElement")}}-Schnittstelle macht nichts. Sie spiegelt das `char`-Attribut des Abschnittselements wider.

> [!NOTE]
> Diese Eigenschaft war dazu gedacht, die Möglichkeit zu unterstützen, Tabellenzellinhalte an einem bestimmten Zeichen (typischerweise dem Dezimalpunkt) auszurichten, wurde aber nie von Browsern implementiert.
>
> Um eine solche Ausrichtung zu erreichen, überprüfen Sie die Unterstützung eines Zeichenfolgenwerts mit der {{cssxref("text-align")}} CSS-Eigenschaft.

## Wert

Ein einzelnes Zeichen.
