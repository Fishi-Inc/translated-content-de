{{APIRef("WebRTC")}}

Die **`remoteId`**-Eigenschaft des
[`RTCOutboundRtpStreamStats`](/de/docs/Web/API/RTCOutboundRtpStreamStats)-Wörterbuchs gibt die
[`id`](/de/docs/Web/API/RTCOutboundRtpStreamStats/id) des [`RTCRemoteInboundRtpStreamStats`](/de/docs/Web/API/RTCRemoteInboundRtpStreamStats)
Objekts an, welches den `RTCRtpReceiver` des entfernten Peers darstellt, der die Medien an den lokalen Peer für diese SSRC sendet.

## Wert

Ein String, der die ID des
[`RTCRemoteInboundRtpStreamStats`](/de/docs/Web/API/RTCRemoteInboundRtpStreamStats)-Objekts enthält, das den `RTCRtpReceiver` des entfernten Peers für die durch dieses Statistikobjekt dargestellte Synchronisationsquelle repräsentiert.
