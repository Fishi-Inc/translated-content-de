{{APIRef}}

Die **`CanvasRenderingContext2D.miterLimit`**-Eigenschaft der Canvas 2D API legt das Miterlimit-Verhältnis fest.

> [!NOTE]
> Weitere Informationen zu Mittern finden Sie unter [Anwenden von Stilen und Farben](/de/docs/Web/API/Canvas_API/Tutorial/Applying_styles_and_colors) im [Canvas-Leitfaden](/de/docs/Web/API/Canvas_API/Tutorial).

## Wert

Eine Zahl, die das Miterlimit-Verhältnis in Koordinatenraumeinheiten angibt. Werte von Null, negativ, {{jsxref("Infinity")}}, und {{jsxref("NaN")}} werden ignoriert. Der Standardwert ist `10.0`.
