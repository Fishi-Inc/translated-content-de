Die schreibgeschützte Eigenschaft **`ProgressEvent.lengthComputable`** ist ein booleanes Flag, das anzeigt, ob die Ressource, die durch das [`ProgressEvent`](/de/docs/Web/API/ProgressEvent) betroffen ist, eine berechenbare Länge hat. Wenn nicht, hat die [`ProgressEvent.total`](/de/docs/Web/API/ProgressEvent/total)-Eigenschaft keinen signifikanten Wert.

## Wert

Ein boolean.
