{{APIRef("WebRTC")}}

Die Eigenschaft **`lastPacketReceivedTimestamp`** des {{domxref("RTCIceCandidatePairStats")}} gibt an, zu welchem Zeitpunkt die Verbindung, die durch das Kandidatenpaar beschrieben wird, zuletzt ein Paket empfangen hat. {{Glossary("STUN")}}-Pakete sind ausgeschlossen.

## Wert

Ein {{domxref("DOMHighResTimeStamp")}}-Objekt, das den Zeitstempel angibt, zu dem die durch das Kandidatenpaar beschriebene Verbindung zuletzt ein Paket empfangen hat, STUN-Pakete ausgeschlossen.
