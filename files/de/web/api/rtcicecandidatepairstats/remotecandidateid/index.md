Die Eigenschaft **`remoteCandidateId`** des [`RTCIceCandidatePairStats`](/de/docs/Web/API/RTCIceCandidatePairStats) ist ein String, der den entfernten [ICE](/de/docs/Glossary/ICE)-Kandidaten eindeutig identifiziert. Dieser Kandidat wurde analysiert, um die [`RTCIceCandidateStats`](/de/docs/Web/API/RTCIceCandidateStats) zu erzeugen, die für die Berechnung der Statistik für dieses Kandidatenpaar verwendet werden.

## Wert

Ein String, der den entfernten [ICE](/de/docs/Glossary/ICE)-Kandidaten eindeutig identifiziert, das heißt, den Kandidaten, der eine Konfiguration für den entfernten Peer beschreibt, der durch das entfernte Ende dieser Statistik dargestellt wird.
