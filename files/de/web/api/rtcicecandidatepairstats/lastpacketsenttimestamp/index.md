{{APIRef("WebRTC")}}

Die Eigenschaft **`lastPacketSentTimestamp`** des {{domxref("RTCIceCandidatePairStats")}} gibt die Zeit an, zu der die durch das Kandidatenpaar beschriebene Verbindung zuletzt ein Paket gesendet hat, ausgenommen {{Glossary("STUN")}}-Pakete.

## Wert

Ein {{domxref("DOMHighResTimeStamp")}}-Objekt, das den Zeitstempel angibt, zu dem die Verbindung des beschriebenen Kandidatenpaars zuletzt ein Paket gesendet hat, STUN-Pakete ausgeschlossen.
