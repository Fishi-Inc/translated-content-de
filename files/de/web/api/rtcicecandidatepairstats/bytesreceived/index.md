{{APIRef("WebRTC")}}

Die **`bytesReceived`**-Eigenschaft des [`RTCIceCandidatePairStats`](/de/docs/Web/API/RTCIceCandidatePairStats)-Wörterbuchs gibt die Gesamtanzahl der Nutzdatenbytes an, die bisher über die Verbindung, die durch das Kandidatenpaar beschrieben wird, empfangen wurden.

## Wert

Ein ganzzahliger Wert, der die Gesamtanzahl der bisher auf der Verbindung, die durch dieses Kandidatenpaar beschrieben wird, empfangenen Bytes angibt.

Es werden nur Datenbytes gezählt; Overhead wie Auffüllung, Header und Ähnliches sind in dieser Zählung nicht enthalten.
