{{APIRef("Geometry Interfaces")}}

Die **`left`** Eigenschaft der **`DOMRectReadOnly`** Schnittstelle gibt den linken Koordinatenwert des `DOMRect` zurück. (Hat denselben Wert wie `x` oder `x + width`, wenn `width` negativ ist.)

## Wert

Ein Doppelwert.
