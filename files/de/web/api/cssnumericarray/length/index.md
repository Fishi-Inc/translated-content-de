{{domxref("CSSNumericArray")}}-Schnittstelle gibt die Anzahl der
{{domxref("CSSNumericValue")}}-Objekte in der Liste zurück.

## Wert

Ein ganzzahliger Wert, der die Anzahl der {{domxref("CSSNumericValue")}}-Objekte in der
Liste darstellt.
