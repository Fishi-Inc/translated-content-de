Der **`CSSMathSum()`**-Konstruktor erstellt ein neues [`CSSMathSum`](/de/docs/Web/API/CSSMathSum)-Objekt, das ein neues [`CSSKeywordValue`](/de/docs/Web/API/CSSKeywordValue)-Objekt erstellt. Dieses Objekt repräsentiert das Ergebnis, das durch den Aufruf von [`add()`](/de/docs/Web/API/CSSNumericValue/add), [`sub()`](/de/docs/Web/API/CSSNumericValue/sub) oder [`toSum()`](/de/docs/Web/API/CSSNumericValue/toSum) auf [`CSSNumericValue`](/de/docs/Web/API/CSSNumericValue) erhalten wird.

## Syntax

```js-nolint
new CSSMathSum(values)
```

### Parameter

- `values`
  - : Eine oder mehrere doppelte Ganzzahlen oder [`CSSNumericValue`](/de/docs/Web/API/CSSNumericValue)-Objekte.
