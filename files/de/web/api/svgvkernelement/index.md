Das **`SVGVKernElement`** Interface entspricht den {{SVGElement("vkern")}}-Elementen.

Ein objektorientierter Zugriff auf die Attribute des {{SVGElement("vkern")}}-Elements über das SVG DOM ist nicht möglich.
