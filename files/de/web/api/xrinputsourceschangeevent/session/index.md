Die Eigenschaft `session` des {{domxref("XRInputSourcesChangeEvent")}} gibt die {{domxref("XRSession")}} an, auf die sich das Ereignis der Änderung der Eingabequellenliste bezieht.

## Wert

Eine {{domxref("XRSession")}}, die die WebXR-Sitzung anzeigt, auf die sich die Änderung der Eingabequellenliste bezieht.
