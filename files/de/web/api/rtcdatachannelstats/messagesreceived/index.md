{{APIRef("WebRTC")}}

Die **`messagesReceived`**-Eigenschaft des {{domxref("RTCDataChannelStats")}} Wörterbuchs gibt die Gesamtanzahl der [`message` Ereignisse](/de/docs/Web/API/RTCDataChannel/message_event) zurück, die für empfangene Nachrichten auf dem zugehörigen {{domxref("RTCDataChannel")}} ausgelöst wurden.

## Wert

Ein positiver Ganzzahlenwert, der die Gesamtanzahl der `message`-Ereignisse für eingehende Daten auf dem zugehörigen Datenkanal angibt.
