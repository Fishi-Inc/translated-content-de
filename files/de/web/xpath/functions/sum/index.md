Die Funktion `sum` gibt eine Zahl zurück, die die Summe der numerischen Werte jedes Knotens in einem gegebenen Knoten-Set ist.

## Syntax

```plain
sum(node-set)
```

### Parameter

- `node-set`
  - : Das zu evaluierende Knoten-Set. Jeder Knoten in diesem Knoten-Set wird so ausgewertet, als wäre er der [number()](/de/docs/Web/XPath/Functions/number) Funktion übergeben worden, und eine Summe der resultierenden Zahlen wird zurückgegeben.

### Rückgabewert

Eine Zahl.

## Beschreibung

Keine.

## Spezifikationen

[XPath 1.0 4.3](https://www.w3.org/TR/1999/REC-xpath-19991116/#function-sum)

Unterstützt.
