Das `<xsl:processing-instruction>`-Element schreibt eine Anweisungsanweisung in das Ausgabedokument.

## Syntax

`<xsl:processing-instruction name=NAME> TEMPLATE </xsl:processing-instruction>`

### Erforderliche Attribute

- `name`
  - : Gibt den Namen dieser Anweisungsanweisung an.

### Optionale Attribute

Keine.

### Typ

Anweisung, erscheint innerhalb einer Vorlage.

## Spezifikationen

XSLT, Abschnitt 7.3

## Gecko-Unterstützung

Unterstützt.
