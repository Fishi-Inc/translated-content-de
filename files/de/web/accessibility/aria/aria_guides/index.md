Accessible Rich Internet Applications (**ARIA**) definieren Möglichkeiten, um das Web für Menschen mit Behinderungen zugänglicher zu machen. Diese Seite listet die Leitfäden auf, die Sie verwenden können, um die Zugänglichkeit von Webseitenfunktionen wie Tabellen, Formularen und Tastaturnavigation zu verbessern.

- [Barrierefreie Formulare gestalten](https://www.w3.org/WAI/tutorials/forms/)
- [Grundlagen von WAI-ARIA verstehen](/de/docs/Learn/Accessibility/WAI-ARIA_basics)
- [Erstellen von tastaturbedienbaren JavaScript-Widgets](/de/docs/Web/Accessibility/Keyboard-navigable_JavaScript_widgets)
- [Verwendung von ARIA für Labels mit eingebetteten Feldern](/de/docs/Web/Accessibility/ARIA/Multipart_labels)
- [Verwaltung der Fokussierung in zusammengesetzten Widgets](https://www.w3.org/WAI/ARIA/apg/practices/keyboard-interface/) (`aria-activedescendant` vs. roving tabindex)
- [Testen von ARIA](https://www.w3.org/WAI/test-evaluate/)
- [Anzeigen barrierefreier Tabellen](https://www.w3.org/WAI/tutorials/tables/)
