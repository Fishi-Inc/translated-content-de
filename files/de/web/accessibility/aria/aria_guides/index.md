Accessible Rich Internet Applications (**ARIA**) definieren Möglichkeiten, das Web für Menschen mit Behinderungen zugänglicher zu machen.
Diese Seite listet die Leitfäden auf, die Sie nutzen können, um die Zugänglichkeit von Webseitenfunktionen wie Tabellen, Formulare und Tastaturnavigation zu verbessern.

- [Gestaltung von barrierefreien Formularen](https://www.w3.org/WAI/tutorials/forms/)
- [Grundlagen von WAI-ARIA verstehen](/de/docs/Learn/Accessibility/WAI-ARIA_basics)
- [Erstellung von tastaturnavigierbaren JavaScript-Widgets](/de/docs/Web/Accessibility/Keyboard-navigable_JavaScript_widgets)
- [Verwendung von ARIA für Labels mit eingebetten Feldern](/de/docs/Web/Accessibility/ARIA/Multipart_labels)
- [Fokusverwaltung in zusammengesetzten Widgets](https://www.w3.org/WAI/ARIA/apg/practices/keyboard-interface/) (`aria-activedescendant` vs. roving tabindex)
- [Testing von ARIA](https://www.w3.org/WAI/test-evaluate/)
- [Anzeige barrierefreier Tabellen](https://www.w3.org/WAI/tutorials/tables/)
