Die `composite` [abstrakte Rolle](/de/docs/Web/Accessibility/ARIA/Roles#6._abstract_roles) zeigt ein Widget an, das navigierbare Nachfahren oder zugehörige Kinder enthalten kann.

> [!NOTE]
> Die `composite` Rolle ist eine abstrakte Rolle. Sie wird hier zur Vollständigkeit der Dokumentation aufgeführt. Sie sollte nicht von Web-Entwicklern verwendet werden.

## Beschreibung

`Composite` ist eine abstrakte Rolle, die für die Ontologie verwendet wird. Verwenden Sie diese Rolle nicht in Inhalten. Verwenden Sie stattdessen die zusammengesetzten Unterklassen von [`grid`](/de/docs/Web/Accessibility/ARIA/Roles/grid_role), [`select`](/de/docs/Web/Accessibility/ARIA/Roles/select_role), [`spinbutton`](/de/docs/Web/Accessibility/ARIA/Roles/spinbutton_role) und [`tablist`](/de/docs/Web/Accessibility/ARIA/Roles/tablist_role).

## Beste Praktiken

Nicht verwenden.
