Der **Anfangswert** einer [CSS](/de/docs/Web/CSS) Eigenschaft ist ihr Standardwert, wie in der Definitionstabelle in der Spezifikation aufgeführt. Die Verwendung des Anfangswertes hängt davon ab, ob eine Eigenschaft vererbt wird oder nicht:

- Bei [vererbten Eigenschaften](/de/docs/Web/CSS/Inheritance#inherited_properties) wird der Anfangswert nur auf das _Wurzelelement_ angewendet, sofern kein [spezifizierter Wert](/de/docs/Web/CSS/specified_value) angegeben ist.
- Bei [nicht vererbten Eigenschaften](/de/docs/Web/CSS/Inheritance#non-inherited_properties) wird der Anfangswert auf _alle Elemente_ angewendet, sofern kein [spezifizierter Wert](/de/docs/Web/CSS/specified_value) angegeben ist.

Sie können den Anfangswert explizit angeben, indem Sie das {{cssxref("initial")}} Schlüsselwort verwenden.

> [!NOTE]
> Der Anfangswert sollte nicht mit dem von dem Browser-Stylesheet angegebenen Wert verwechselt werden.
