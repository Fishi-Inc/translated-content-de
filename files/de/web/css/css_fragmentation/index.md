Das Modul **CSS Fragmentierung** definiert, wie Inhalte dargestellt werden, wenn sie über mehrere [Seiten](/de/docs/Web/CSS/CSS_paged_media), Regionen oder [Spalten](/de/docs/Web/CSS/CSS_multicol_layout) gebrochen (fragmentiert) werden.

Fragmentierung tritt auf, wenn ein Inline-Box-Element auf mehrere Linien umbrochen wird. Es tritt auch auf, wenn ein Block mehr als eine Spalte innerhalb eines Spalten-Layout-Containers umfasst, oder beim Drucken einen Seitenumbruch umfasst. Jedes Stück der Darstellung des Elements wird als _Fragment_ bezeichnet.
