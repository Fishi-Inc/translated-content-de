Die **`:-moz-handler-crashed`** [CSS](/de/docs/Web/CSS) [Pseudoklasse](/de/docs/Web/CSS/Pseudo-classes) ist eine [Mozilla-Erweiterung](/de/docs/Web/CSS/Mozilla_Extensions), die Elemente auswählt, die nicht angezeigt werden können, weil das für deren Darstellung verantwortliche Plugin abgestürzt ist.

> [!NOTE]
> Dieser Selektor ist hauptsächlich für Theme-Entwickler gedacht.

## Syntax

```css
:-moz-handler-crashed {
  /* ... */
}
```

## Spezifikationen

Nicht Teil eines Standards.
