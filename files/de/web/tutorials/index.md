Die Links auf dieser Seite führen zu einer Vielzahl von Tutorials und Schulungsmaterialien. Egal, ob Sie gerade erst anfangen, die Grundlagen lernen oder ein alter Hase in der Webentwicklung sind, hier finden Sie hilfreiche Ressourcen für Best Practices.

Diese Ressourcen werden von fortschrittlichen Unternehmen und Webentwicklern erstellt, die offene Standards und Best Practices für die Webentwicklung übernommen haben und durch eine offene Inhaltslizenz wie Creative Commons Übersetzungen bereitstellen oder erlauben.

## Für komplette Anfänger im Web

- [Erste Schritte mit dem Web](/de/docs/Learn/Getting_started_with_the_web)
  - : _Erste Schritte mit dem Web_ ist eine kurze Serie, die Sie mit den praktischen Aspekten der Webentwicklung vertraut macht. Sie richten die Tools ein, die Sie benötigen, um eine einfache Webseite zu erstellen, und veröffentlichen Ihren eigenen einfachen Code.

## HTML Tutorials

### Einführungsniveau

- [Einführung in HTML](/de/docs/Learn/HTML/Introduction_to_HTML)
  - : Dieses Modul bereitet das Feld vor, indem es Sie an wichtige Konzepte und die Syntax gewöhnt, HTML auf Text anwendet, zeigt, wie Hyperlinks erstellt werden, und wie HTML verwendet werden kann, um eine Webseite zu strukturieren.
- [MDN HTML Elementreferenz](/de/docs/Web/HTML/Element)
  - : Eine umfassende Referenz für HTML-Elemente und wie die verschiedenen Browser sie unterstützen.
- [Eine einfache Webseite mit HTML erstellen](https://www.theblogstarter.com/html-for-beginners/)
  - : Ein HTML-Leitfaden für Anfänger, der Erklärungen zu gängigen Tags einschließlich HTML-Tags enthält. Außerdem eine Schritt-für-Schritt-Anleitung zum Erstellen einer einfachen Webseite mit Codebeispielen.
- [HTML Herausforderungen](https://en.wikiversity.org/wiki/Web_Design/HTML_Challenges)
  - : Nutzen Sie diese Herausforderungen, um Ihre HTML-Fähigkeiten zu verbessern (zum Beispiel "Sollte ich ein \<h2>-Element oder ein \<strong>-Element verwenden?"), mit Fokus auf bedeutungsvolles Markup.

### Mittleres Niveau

- [Multimedia und Einbetten](/de/docs/Learn/HTML/Multimedia_and_embedding)
  - : Dieses Modul erforscht, wie HTML verwendet wird, um Multimedia in Ihre Webseiten einzubinden, einschließlich der verschiedenen Möglichkeiten, wie Bilder eingebunden werden können, und wie Videos, Audios und sogar ganze andere Webseiten eingebettet werden können.
- [HTML-Tabellen](/de/docs/Learn/HTML/Tables)
  - : Tabellendaten auf einer Webseite verständlich und [zugänglich](/de/docs/Glossary/Accessibility) darzustellen, kann eine Herausforderung sein. Dieses Modul deckt das grundlegende Tabellen-Markup ab, sowie komplexere Funktionen wie die Implementierung von Beschriftungen und Zusammenfassungen.

### Fortgeschrittenes Niveau

- [HTML-Formulare](/de/docs/Learn/Forms)
  - : Formulare sind ein sehr wichtiger Teil des Webs — sie bieten viel von der Funktionalität, die Sie zum Interagieren mit Webseiten benötigen, z.B. Registrierung und Anmeldung, Feedback senden, Produkte kaufen und mehr. Dieses Modul bringt Ihnen bei, wie Sie die clientseitigen Teile von Formularen erstellen.
- [Tipps zum Verfassen von schnell ladenden HTML-Seiten](/de/docs/Learn/HTML/Howto/Author_fast-loading_HTML_pages)
  - : Optimieren Sie Webseiten, um eine reaktionsschnellere Seite für Besucher zu bieten und die Belastung Ihres Webservers und der Internetverbindung zu reduzieren.

## CSS Tutorials

### Einführungsniveau

- [CSS-Grundlagen](/de/docs/Learn/Getting_started_with_the_web/CSS_basics)
  - : CSS (Cascading Style Sheets) ist der Code, mit dem Sie Ihre Webseite gestalten. _CSS-Grundlagen_ führt Sie durch das, was Sie benötigen, um loszulegen. Wir beantworten Fragen wie: Wie mache ich meinen Text schwarz oder rot? Wie bringe ich meine Inhalte an bestimmten Stellen auf dem Bildschirm zum Vorschein? Wie schmücke ich meine Webseite mit Hintergrundbildern und -farben?
- [CSS-Erste Schritte](/de/docs/Learn/CSS/First_steps)
  - : CSS (Cascading Style Sheets) wird verwendet, um Webseiten zu gestalten und zu layouten — zum Beispiel, um die Schriftart, Farbe, Größe und Abstände Ihrer Inhalte zu ändern, diese in mehrere Spalten aufzuteilen oder Animationen und andere dekorative Funktionen hinzuzufügen. Dieses Modul bietet Ihnen einen sanften Einstieg in den Weg zur CSS-Meisterschaft mit den Grundlagen, wie es funktioniert, wie die Syntax aussieht und wie Sie damit beginnen können, HTML zu stylen.
- [CSS-Bausteine](/de/docs/Learn/CSS/Building_blocks)

  - : Dieses Modul führt weiter, wo [CSS-Erste Schritte](/de/docs/Learn/CSS/First_steps) aufgehört hat — jetzt, da Sie sich mit der Sprache und ihrer Syntax vertraut gemacht haben und einige Grundkenntnisse im Umgang damit erworben haben, ist es an der Zeit, etwas tiefer einzutauchen. Dieses Modul befasst sich mit dem Kaskaden- und Vererbungsprinzip, allen verfügbaren Selektortypen, Einheiten, Größen, dem Styling von Hintergründen und Rändern, Debugging und vielem mehr.

    Das Ziel ist es, Ihnen ein Toolkit zur Verfügung zu stellen, um kompetentes CSS zu schreiben und Ihnen alle grundlegenden Theorien zu verstehen, bevor Sie sich spezifischeren Disziplinen wie [Textgestaltung](/de/docs/Learn/CSS/Styling_text) und [CSS-Layout](/de/docs/Learn/CSS/CSS_layout) zuwenden.

- [Selektoren](/de/docs/Learn/CSS/Building_blocks/Selectors)

  - : HTML-Elemente, einschließlich basierend auf dem Zustand des Elements, mit CSS anvisieren.

- [Spezifität](/de/docs/Web/CSS/Specificity)

  - : Verstehen Sie den Browser-Algorithmus, der bestimmt, welche CSS-Deklarationen auf ein Element angewendet werden, wenn es konkurrierende Deklarationen gibt, mit einem [Spezifität-Quiz](https://estelle.github.io/CSS/selectors/exercises/specificity.html)

- [Kaskade & Vererbung](/de/docs/Learn/CSS/Building_blocks/Cascade_and_inheritance)

  - : Die Kaskade, Spezifität und Vererbung steuern, wie CSS auf HTML angewendet wird und wie Konflikte zwischen Stil-Deklarationen gelöst werden.

- [Text gestalten](/de/docs/Learn/CSS/Styling_text)
  - : Hier betrachten wir die Grundlagen der Textgestaltung, einschließlich der Einstellung von Schriftart, Fettdruck und Kursivschrift, Zeilen- und Zeichenabstand sowie Schlagschatten und andere Textmerkmale. Wir beenden das Modul, indem wir uns ansehen, wie benutzerdefinierte Schriftarten auf Ihrer Seite angewendet werden und Listen und Links gestylt werden.
- [Häufige CSS-Fragen](/de/docs/Learn/CSS/Howto/CSS_FAQ)
  - : Häufige Fragen und Antworten für Anfänger.

### Mittleres Niveau

- [CSS-Layout](/de/docs/Learn/CSS/CSS_layout)
  - : An diesem Punkt haben wir bereits die CSS-Grundlagen, das Stylen von Text und das Stylen und Manipulieren der Boxen, auf denen Ihre Inhalte sitzen, betrachtet. Jetzt ist es an der Zeit, zu schauen, wie Sie Ihre Boxen im Verhältnis zum Viewport und zueinander platzieren. Wir haben die notwendigen Voraussetzungen behandelt und können jetzt tief in das CSS-Layout eintauchen und uns verschiedene Anzeigeeinstellungen, traditionelle Layoutmethoden mit Float und Positionierung sowie neue Layout-Tools wie Flexbox ansehen.
- [CSS-Referenz](/de/docs/Web/CSS/Reference)
  - : Vollständige Referenz zu CSS mit Details zur Unterstützung durch Firefox und andere Browser.
- [Flüssige Raster](/en.wikipedia.org/wiki/Fluid_grid)
  - : Entwerfen Sie Layouts, die sich mit dem Browserfenster flüssig ändern, während Sie immer noch ein typografisches Raster verwenden.
- [CSS Herausforderungen](https://en.wikiversity.org/wiki/Web_Design/CSS_challenges)
  - : Beweisen Sie Ihre CSS-Fähigkeiten und sehen Sie, wo Sie noch Übung benötigen.

### Fortgeschrittenes Niveau

- [Verwendung von CSS-Transformationen](/de/docs/Web/CSS/CSS_transforms/Using_CSS_transforms)
  - : Rotation, Schrägung, Skalierung und Translation mit CSS anwenden.
- [CSS-Übergänge](/de/docs/Web/CSS/CSS_transitions/Using_CSS_transitions)
  - : CSS-Übergänge bieten eine Möglichkeit, Änderungen an CSS-Eigenschaften zu animieren, anstatt die Änderungen sofort wirksam werden zu lassen.
- [Canvas-Tutorial](/de/docs/Web/API/Canvas_API/Tutorial)
  - : Lernen Sie, wie Sie Grafiken mit Skripten mithilfe des `<canvas>`-Elements zeichnen.

Diese Dinge haben Sie im Kurs bereits gesehen, aber nur flüchtig — hier werden wir alles explizit erläutern.

### Mittleres Niveau

- [Einführung in JavaScript-Objekte](/de/docs/Learn/JavaScript/Objects)
  - : In JavaScript sind die meisten Dinge Objekte, von den Kernfunktionen von JavaScript wie Strings und Arrays bis zu den Browser-APIs, die auf JavaScript aufbauen. Sie können sogar Ihre eigenen Objekte erstellen, um verwandte Funktionen und Variablen in effiziente Pakete zu kapseln. Das objektorientierte Wesen von JavaScript ist wichtig zu verstehen, wenn Sie Ihr Wissen über die Sprache vertiefen und effizienteren Code schreiben möchten, daher haben wir dieses Modul bereitgestellt, um Ihnen zu helfen. Hier lehren wir die Objekttheorie und -syntax im Detail, zeigen, wie Sie Ihre eigenen Objekte erstellen, und erklären, was JSON-Daten sind und wie man damit arbeitet.
- [Clientseitige Web-APIs](/de/docs/Learn/JavaScript/Client-side_web_APIs)
  - : Beim Schreiben von clientseitigem JavaScript für Webseiten oder Anwendungen werden Sie nicht weit kommen, bevor Sie beginnen, APIs zu verwenden — Schnittstellen zur Manipulation verschiedener Aspekte des Browsers und Betriebssystems, auf dem die Seite läuft, oder sogar Daten von anderen Webseiten oder Diensten. In diesem Modul werden wir erforschen, was APIs sind, und wie man einige der häufigsten APIs nutzt, die Ihnen in Ihrer Entwicklungsarbeit häufig begegnen werden.
- [Eloquent JavaScript](https://eloquentjavascript.net/)
  - : Ein umfassender Leitfaden zu mittleren und fortgeschrittenen JavaScript-Methoden.
- [Speaking JavaScript](https://exploringjs.com/es5/)
  - : Für Programmierer, die JavaScript schnell und richtig lernen wollen, und für JavaScript-Programmierer, die ihre Fähigkeiten vertiefen und/oder bestimmte Themen nachschlagen wollen.
- [Essenzielle JavaScript-Entwurfsmuster](https://addyosmani.com/resources/essentialjsdesignpatterns/book/)

### Erweiterungsentwicklung

- [WebExtensions](/de/docs/Mozilla/Add-ons/WebExtensions)
  - : WebExtensions ist ein systemübergreifendes System zur Entwicklung von Browser-Add-ons. In großem Umfang ist das System kompatibel mit der von Google Chrome und Opera unterstützten [Erweiterungs-API](https://developer.chrome.com/docs/extensions/reference/). Erweiterungen, die für diese Browser geschrieben wurden, werden in den meisten Fällen in Firefox oder [Microsoft Edge](https://learn.microsoft.com/en-us/archive/microsoft-edge/legacy/developer/) mit [nur wenigen Änderungen](https://extensionworkshop.com/documentation/develop/porting-a-google-chrome-extension/) funktionieren. Die API ist auch vollständig kompatibel mit [multiprozess Firefox](https://wiki.mozilla.org/Firefox/multiprocess).
