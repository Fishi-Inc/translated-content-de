{{HTMLSidebar("HTML_Elements")}}

Diese Seite listet alle {{Glossary("HTML", "HTML")}} {{Glossary("Element", "Elemente")}} auf, die mit {{Glossary("Tag", "Tags")}} erstellt werden.

Sie sind nach Funktion gruppiert, um Ihnen die Orientierung zu erleichtern. Eine alphabetische Liste aller Elemente finden Sie in der Seitenleiste auf jeder Elementseite sowie auf dieser Seite.

> [!NOTE]
> Weitere Informationen zu den Grundlagen von HTML-Elementen und -Attributen finden Sie in [dem Abschnitt über Elemente im Artikel Einführung in HTML](/de/docs/Learn/HTML/Introduction_to_HTML/Getting_started#anatomy_of_an_html_element).

## Hauptwurzel

| Element                 | Beschreibung                                                                                                                                                                       |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("html")}} | Stellt die Wurzel (das oberste Element) eines HTML-Dokuments dar und wird daher auch als _Wurzelelement_ bezeichnet. Alle anderen Elemente müssen Nachfolger dieses Elements sein. |

## Dokument-Metadaten

Metadaten enthalten Informationen über die Seite. Dazu gehören Informationen über Stile, Skripte und Daten, die Software wie {{Glossary("search_engine", "Suchmaschinen")}}, {{Glossary("Browser", "Browser")}} usw. bei der Verwendung und Darstellung der Seite helfen. Metadaten für Stile und Skripte können auf der Seite definiert oder in eine andere Datei verlinkt werden, die diese Informationen enthält.

| Element                  | Beschreibung                                                                                                                                                                                                                                                                                                                     |
| ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("base")}}  | Gibt die Basis-URL an, die für alle relativen URLs in einem Dokument verwendet werden soll. Es kann nur ein solches Element in einem Dokument geben.                                                                                                                                                                             |
| {{HTMLElement("head")}}  | Enthält maschinenlesbare Informationen (Metadaten) über das Dokument, wie seinen [Titel](/de/docs/Web/HTML/Element/title), [Skripte](/de/docs/Web/HTML/Element/script) und [Stylesheets](/de/docs/Web/HTML/Element/style).                                                                                                       |
| {{HTMLElement("link")}}  | Gibt Beziehungen zwischen dem aktuellen Dokument und einer externen Ressource an. Dieses Element wird am häufigsten verwendet, um auf CSS zu verlinken, kann aber auch verwendet werden, um Sitzeigenschaften (sowohl "Favoriten" Stilarten als auch Symbole für den Startbildschirm und Apps auf mobilen Geräten) einzurichten. |
| {{HTMLElement("meta")}}  | Stellt {{Glossary("Metadata", "Metadaten")}} dar, die nicht durch andere HTML-Elemente im Zusammenhang mit Metadaten dargestellt werden können, wie {{HTMLElement("base")}}, {{HTMLElement("link")}}, {{HTMLElement("script")}}, {{HTMLElement("style")}} und {{HTMLElement("title")}}.                                          |
| {{HTMLElement("style")}} | Enthält Stilinformationen für ein Dokument oder einen Teil eines Dokuments. Es enthält CSS, das auf den Inhalt des Dokuments angewendet wird, das dieses Element enthält.                                                                                                                                                        |
| {{HTMLElement("title")}} | Definiert den Titel des Dokuments, der in der Titelleiste eines {{Glossary("Browser", "Browsers")}} oder in einem Tab einer Seite angezeigt wird. Es enthält nur Text; HTML-Tags innerhalb des Elements werden, falls vorhanden, ebenfalls als reiner Text behandelt.                                                            |

| ----------------------- | ---------------------------------------------------------------------------------------------------- |
| {{HTMLElement("body")}} | Stellt den Inhalt eines HTML-Dokuments dar. Es kann nur ein solches Element in einem Dokument geben. |

## Inhaltliche Gliederung

Inhaltliche Gliederungselemente ermöglichen es Ihnen, den Inhalt des Dokuments in logische Teile zu organisieren. Verwenden Sie die Gliederungselemente, um eine breite Struktur für den Seiteninhalt zu schaffen, einschließlich Header- und Footer-Navigation und Überschriftselemente, um Abschnitte des Inhalts zu identifizieren.

| Element                                                                                                                                                                                                                                                                                                                            | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("address")}}                                                                                                                                                                                                                                                                                                         | Gibt an, dass der umschlossene HTML-Text Kontaktinformationen für eine Person oder Personen oder eine Organisation liefert.                                                                                                                                                                                                                                                                                                                            |
| {{HTMLElement("article")}}                                                                                                                                                                                                                                                                                                         | Stellt eine eigenständige Komposition in einem Dokument, einer Seite, einer Anwendung oder einer Website dar, die unabhängig verteilt oder wiederverwendet werden soll (z. B. in einer Syndikation). Beispiele umfassen einen Forumsbeitrag, einen Magazin- oder Zeitungsartikel, einen Blogeintrag, eine Produktkarte, einen von Benutzern übermittelten Kommentar, ein interaktives Widget oder Gadget oder jedes andere unabhängige Inhaltselement. |
| {{HTMLElement("aside")}}                                                                                                                                                                                                                                                                                                           | Stellt einen Teil eines Dokuments dar, dessen Inhalt nur indirekt mit dem Hauptinhalt des Dokuments verwandt ist. Seitenhinweise werden häufig als Seitenleisten oder Hinweisboxen dargestellt.                                                                                                                                                                                                                                                        |
| {{HTMLElement("footer")}}                                                                                                                                                                                                                                                                                                          | Stellt einen Footer für seinen nächsten Vorfahren im [Gliederungsinhalt](/de/docs/Web/HTML/Content_categories#sectioning_content) oder Element der [Gliederungswurzel](/de/docs/Web/HTML/Element/Heading_Elements) dar. Ein `<footer>` enthält typischerweise Informationen über den Autor des Abschnitts, Urheberrechtsdaten oder Links zu verwandten Dokumenten.                                                                                     |
| {{HTMLElement("header")}}                                                                                                                                                                                                                                                                                                          | Stellt einleitenden Inhalt dar, typischerweise eine Gruppe von einleitenden oder navigationsbezogenen Hilfsmitteln. Es kann einige Überschriftselemente enthalten, aber auch ein Logo, ein Suchformular, einen Autorennamen und andere Elemente.                                                                                                                                                                                                       |
| [`<h1>`](/de/docs/Web/HTML/Element/Heading_Elements), [`<h2>`](/de/docs/Web/HTML/Element/Heading_Elements), [`<h3>`](/de/docs/Web/HTML/Element/Heading_Elements), [`<h4>`](/de/docs/Web/HTML/Element/Heading_Elements), [`<h5>`](/de/docs/Web/HTML/Element/Heading_Elements), [`<h6>`](/de/docs/Web/HTML/Element/Heading_Elements) | Stellen sechs Ebenen von Abschnittsüberschriften dar. `<h1>` ist die höchste Abschnittsebene und `<h6>` ist die niedrigste.                                                                                                                                                                                                                                                                                                                            |
| {{HTMLElement("hgroup")}}                                                                                                                                                                                                                                                                                                          | Stellt eine Überschrift dar, die mit sekundären Inhalten gruppiert ist, wie z.B. Untertitel, einem alternativen Titel oder einem Schlagwort.                                                                                                                                                                                                                                                                                                           |
| {{HTMLElement("main")}}                                                                                                                                                                                                                                                                                                            | Stellt den dominanten Inhalt des Körpers eines Dokuments dar. Der Hauptinhaltbereich besteht aus Inhalten, die direkt mit dem Hauptthema eines Dokuments oder der zentralen Funktionalität einer Anwendung in Verbindung stehen oder diese erweitern.                                                                                                                                                                                                  |
| {{HTMLElement("nav")}}                                                                                                                                                                                                                                                                                                             | Stellt einen Abschnitt einer Seite dar, dessen Zweck es ist, Navigationslinks bereitzustellen, entweder innerhalb des aktuellen Dokuments oder zu anderen Dokumenten. Häufige Beispiele für Navigationsabschnitte sind Menüs, Inhaltsverzeichnisse und Indizes.                                                                                                                                                                                        |
| {{HTMLElement("section")}}                                                                                                                                                                                                                                                                                                         | Stellt einen generischen eigenständigen Abschnitt eines Dokuments dar, für den es kein spezifischeres semantisches Element gibt. Abschnitte sollten immer eine Überschrift haben, mit sehr wenigen Ausnahmen.                                                                                                                                                                                                                                          |
| {{HTMLElement("search")}}                                                                                                                                                                                                                                                                                                          | Stellt einen Teil dar, der eine Menge von Formulareingabefeldern oder anderen Inhalten enthält, die sich auf die Durchführung einer Suche oder einer Filteroperation beziehen.                                                                                                                                                                                                                                                                         |

## Textinhalt

Verwenden Sie HTML-Textinhaltselemente, um Blöcke oder Abschnitte von Inhalten zu organisieren, die zwischen den öffnenden {{HTMLElement("body")}} und schließenden `</body>`-Tags platziert sind. Diese Elemente sind wichtig für die {{Glossary("accessibility", "Barrierefreiheit")}} und {{Glossary("SEO", "SEO")}} und helfen, den Zweck oder die Struktur dieses Inhalts zu identifizieren.

| Element                       | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("blockquote")}} | Gibt an, dass der umschlossene Text ein erweitertes Zitat ist. Üblicherweise wird dies visuell durch Einrückung dargestellt. Eine URL für die Quelle des Zitats kann mit dem `cite`-Attribut angegeben werden, während eine Textdarstellung der Quelle mit dem {{HTMLElement("cite")}}-Element angegeben werden kann.                                                                                                                                                               |
| {{HTMLElement("dd")}}         | Liefert die Beschreibung, Definition oder den Wert für den vorhergehenden Begriff ({{HTMLElement("dt")}}) in einer Definitionsliste ({{HTMLElement("dl")}}).                                                                                                                                                                                                                                                                                                                        |
| {{HTMLElement("div")}}        | Der generische Container für Flussinhalte. Es hat keinen Einfluss auf den Inhalt oder das Layout, bis es auf irgendeine Weise mit CSS gestylt wird (z.B. Styling wird direkt darauf angewendet oder eine Layoutmodell wie {{Glossary("Flexbox", "Flexbox")}} wird auf sein Elternelement angewendet).                                                                                                                                                                               |
| {{HTMLElement("dl")}}         | Stellt eine Definitionsliste dar. Das Element schließt eine Liste von Gruppen von Begriffen ein (angegeben durch das {{HTMLElement("dt")}}-Element) und Beschreibungen (bereitgestellt durch {{HTMLElement("dd")}}-Elemente). Häufige Verwendungen dieses Elements bestehen darin, ein Glossar zu implementieren oder Metadaten anzuzeigen (eine Liste von Schlüssel-Wert-Paaren).                                                                                                  |
| {{HTMLElement("dt")}}         | Gibt einen Begriff in einer Definitions- oder Erklärungsliste an und muss daher innerhalb eines {{HTMLElement("dl")}}-Elements verwendet werden. Es folgt normalerweise einem {{HTMLElement("dd")}}-Element; jedoch geben mehrere `<dt>`-Elemente in einer Reihe mehrere Begriffe an, die alle durch das unmittelbar folgende {{HTMLElement("dd")}}-Element definiert werden.                                                                                                       |
| {{HTMLElement("figcaption")}} | Stellt eine Bildunterschrift oder Legende dar, die den Rest des Inhalts seines Elternelements {{HTMLElement("figure")}} beschreibt.                                                                                                                                                                                                                                                                                                                                                 |
| {{HTMLElement("figure")}}     | Stellt eigenständigen Inhalt dar, der potenziell mit einer optionalen Bildunterschrift versehen ist, die mit dem {{HTMLElement("figcaption")}}-Element angegeben wird. Die Abbildung, ihre Bildunterschrift und ihre Inhalte werden als eine Einheit referenziert.                                                                                                                                                                                                                  |
| {{HTMLElement("hr")}}         | Stellt einen thematischen Bruch zwischen Elementen auf Absatzebene dar: zum Beispiel eine Änderung der Szene in einer Geschichte oder ein Themenwechsel innerhalb eines Abschnitts.                                                                                                                                                                                                                                                                                                 |
| {{HTMLElement("li")}}         | Stellt ein Element in einer Liste dar. Es muss in einem übergeordneten Element enthalten sein: einer geordneten Liste ({{HTMLElement("ol")}}), einer ungeordneten Liste ({{HTMLElement("ul")}}) oder einem Menü ({{HTMLElement("menu")}}). In Menüs und ungeordneten Listen werden Listenelemente normalerweise mit Aufzählungszeichen dargestellt. In geordneten Listen werden sie normalerweise mit einem aufsteigenden Zähler links angezeigt, wie eine Zahl oder ein Buchstabe. |
| {{HTMLElement("menu")}}       | Eine semantische Alternative zu {{HTMLElement("ul")}}, wird jedoch von Browsern (und über den Accessibility-Tree) wie {{HTMLElement("ul")}} behandelt. Es stellt eine ungeordnete Liste von Elementen dar (die durch {{HTMLElement("li")}}-Elemente repräsentiert werden).                                                                                                                                                                                                          |
| {{HTMLElement("ol")}}         | Stellt eine geordnete Liste von Elementen dar — typischerweise als nummerierte Liste angezeigt.                                                                                                                                                                                                                                                                                                                                                                                     |
| {{HTMLElement("p")}}          | Stellt einen Absatz dar. Absätze werden in visuellen Medien üblicherweise als Textblöcke dargestellt, die durch Leerzeilen und/oder Einrückung der ersten Zeile von benachbarten Blöcken getrennt sind, aber HTML-Absätze können jede strukturelle Gruppierung verwandter Inhalte sein, wie z.B. Bilder oder Formularfelder.                                                                                                                                                        |
| {{HTMLElement("pre")}}        | Stellt vorformatierten Text dar, der genau so dargestellt werden soll, wie er in der HTML-Datei geschrieben steht. Der Text wird typischerweise mit einer nicht proportionalen oder [monospace](https://en.wikipedia.org/wiki/Monospaced_font) Schriftart angezeigt. Leerzeichen innerhalb dieses Elements werden so angezeigt, wie sie geschrieben sind.                                                                                                                           |
| {{HTMLElement("ul")}}         | Stellt eine ungeordnete Liste von Elementen dar, typischerweise als Aufzählungsliste angezeigt.                                                                                                                                                                                                                                                                                                                                                                                     |

## Inline-Text-Semantik

Verwenden Sie die HTML-Inline-Text-Semantik, um die Bedeutung, Struktur oder den Stil eines Wortes, einer Zeile oder eines beliebigen anderen Textstückes zu definieren.

| Element                   | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("a")}}      | Erstellt zusammen mit seinem `href`-Attribut einen Hyperlink zu Webseiten, Dateien, E-Mail-Adressen, Positionen innerhalb der aktuellen Seite oder allem anderen, was eine URL adressieren kann.                                                                                                                                                                                                                                                                                                                                                                                                      |
| {{HTMLElement("abbr")}}   | Stellt eine Abkürzung oder ein Akronym dar.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| {{HTMLElement("b")}}      | Wird verwendet, um die Aufmerksamkeit des Lesers auf den Inhalt des Elements zu lenken, dem ansonsten keine besondere Bedeutung zuerkannt wird. Dies wurde früher als Fettschrift-Element bezeichnet, und die meisten Browser stellen den Text immer noch in Fettdruck dar. Sie sollten `<b>` jedoch nicht verwenden, um Text zu stylen oder um Bedeutung zu verleihen. Wenn Sie Fettschrifttext erstellen möchten, sollten Sie die CSS-Eigenschaft {{cssxref("font-weight")}} verwenden. Wenn Sie angeben möchten, dass ein Element besonders wichtig ist, sollten Sie das strong-Element verwenden. |
| {{HTMLElement("bdi")}}    | Weist den Bidirektionalen Algorithmus des Browsers an, den darin enthaltenen Text isoliert vom umgebenden Text zu behandeln. Es ist besonders nützlich, wenn eine Website dynamisch Text einfügt und die Richtung des eingefügten Textes nicht bekannt ist.                                                                                                                                                                                                                                                                                                                                           |
| {{HTMLElement("bdo")}}    | Überschreibt die aktuelle Textrichtung, sodass der Text innerhalb in einer anderen Richtung gerendert wird.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| {{HTMLElement("br")}}     | Erzeugt einen Zeilenumbruch im Text (Carriage-Return). Es ist nützlich zum Schreiben eines Gedichts oder einer Adresse, bei denen die Aufteilung der Zeilen von Bedeutung ist.                                                                                                                                                                                                                                                                                                                                                                                                                        |
| {{HTMLElement("cite")}}   | Wird verwendet, um den Titel eines zitierten kreativen Werkes zu markieren. Der Verweis kann in abgekürzter Form gemäß den kontextgemäßen Konventionen in Bezug auf Zitationsmetadaten angegeben werden.                                                                                                                                                                                                                                                                                                                                                                                              |
| {{HTMLElement("code")}}   | Zeigt seinen Inhalt auf eine Weise an, die darauf hinweist, dass der Text ein kurzes Fragment von Computercode ist. Standardmäßig wird der Inhaltstext mit der standardmäßigen nichtproportionalen Schriftart des Benutzeragents angezeigt.                                                                                                                                                                                                                                                                                                                                                           |
| {{HTMLElement("data")}}   | Verknüpft ein bestimmtes Inhaltselement mit einer maschinenlesbaren Übersetzung. Wenn der Inhalt zeit- oder datumsbezogen ist, muss das `<time>`-Element verwendet werden.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| {{HTMLElement("dfn")}}    | Wird verwendet, um den Begriff hervorzuheben, der im Kontext einer Definitionsphrase oder eines Satzes definiert wird. Das übergeordnete {{HTMLElement("p")}}-Element, das {{HTMLElement("dt")}}/{{HTMLElement("dd")}}-Paar oder der nächste Abschnittsvorfahre des `<dfn>`-Elements wird als Definition des Begriffs angesehen.                                                                                                                                                                                                                                                                      |
| {{HTMLElement("em")}}     | Markiert Text, der eine Betonung hat. Das `<em>`-Element kann verschachtelt werden, wobei jede Verschachtelungsebene einen höheren Grad an Betonung anzeigt.                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| {{HTMLElement("i")}}      | Stellt einen Bereich von Text dar, der aus irgendeinem Grund vom normalen Text abgehoben ist, wie zum Beispiel idiomatischer Text, Fachbegriffe und Taxonomie-Bezeichnungen. Historisch gesehen wurden diese mit kursiver Schrift präsentiert, was die ursprüngliche Quelle der Benennung dieses Elements mit `<i>` ist.                                                                                                                                                                                                                                                                              |
| {{HTMLElement("kbd")}}    | Stellt einen Bereich von Inline-Text dar, der eine Benutzereingabe von der Tastatur, Spracheingabe oder einem anderen Texteingabegerät darstellt. Üblicherweise stellt der Benutzeragent den Inhalt eines `<kbd>`-Elements mit seiner standardmäßigen nichtproportionalen Schriftart dar, obwohl dies nicht durch den HTML-Standard vorgeschrieben ist.                                                                                                                                                                                                                                               |
| {{HTMLElement("mark")}}   | Stellt Text dar, der zur Verweis- oder Notationszwecken markiert oder hervorgehoben wird, aufgrund der Relevanz der markierten Passage im umgebenden Kontext.                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("q")}}      | Gibt an, dass der umschlossene Text ein kurzes Inline-Zitat ist. Die meisten modernen Browser implementieren dies, indem sie den Text in Anführungszeichen setzen. Dieses Element ist für kurze Zitate gedacht, die keinen Absatzumbruch erfordern; für lange Zitate verwenden Sie das {{HTMLElement("blockquote")}}-Element.                                                                                                                                                                                                                                                                         |
| {{HTMLElement("rp")}}     | Wird verwendet, um Fallback-Klammern für Browser bereitzustellen, die die Anzeige von Ruby-Anmerkungen mit dem {{HTMLElement("ruby")}}-Element nicht unterstützen. Ein `<rp>`-Element sollte jedes der öffnenden und schließenden Klammern einschließen, die das {{HTMLElement("rt")}}-Element umgeben, das den Text der Annotation enthält.                                                                                                                                                                                                                                                          |
| {{HTMLElement("rt")}}     | Gibt die Ruby-Textkomponente einer Ruby-Anmerkung an, die dazu verwendet wird, Informationen zur Aussprache, Übersetzung oder Transkription für ostasiatische Typografie bereitzustellen. Das `<rt>`-Element muss immer innerhalb eines {{HTMLElement("ruby")}}-Elements enthalten sein.                                                                                                                                                                                                                                                                                                              |
| {{HTMLElement("ruby")}}   | Stellt kleine Annotationen dar, die oberhalb, unterhalb oder neben dem Basistext gerendert werden und üblicherweise verwendet werden, um die Aussprache von ostasiatischen Zeichen darzustellen. Es kann auch zur Annotation anderer Arten von Text verwendet werden, obwohl diese Verwendung weniger verbreitet ist.                                                                                                                                                                                                                                                                                 |
| {{HTMLElement("s")}}      | Rendert Text mit Durchstreichung oder einer Linie durch den Text. Verwenden Sie das `<s>`-Element, um Dinge darzustellen, die nicht mehr relevant oder korrekt sind. `<s>` ist jedoch nicht geeignet, um Dokumentänderungen anzuzeigen; hierzu verwenden Sie die {{HTMLElement("del")}}- und {{HTMLElement("ins")}}-Elemente, je nachdem, was geeignet ist.                                                                                                                                                                                                                                           |
| {{HTMLElement("samp")}}   | Wird verwendet, um Inline-Text einzuschließen, der eine Probe (oder ein zitiertes) Ausgabe eines Computerprogramms darstellt. Sein Inhalt wird typischerweise mit der standardmäßigen nichtproportionalen Schriftart des Browsers (wie [Courier](<https://en.wikipedia.org/wiki/Courier_(typeface)>) oder Lucida Console) dargestellt.                                                                                                                                                                                                                                                                |
| {{HTMLElement("small")}}  | Stellt Nebenkommentare und Kleindruck dar, wie Beispielsweise Urheberrechte und gesetzliche Bestimmungen, unabhängig von seiner als Präsentation gestylten Darstellung dar. Standardmäßig rendert es den Text darin um eine Schriftgröße kleiner, wie z.B. von `small` zu `x-small`.                                                                                                                                                                                                                                                                                                                  |
| {{HTMLElement("span")}}   | Ein generischer Inline-Container für Phraseninhalt, der von Haus aus nichts repräsentiert. Es kann verwendet werden, um Elemente zu gruppieren, die für Styling-Zwecke gedacht sind (mit den Attributen `class` oder `id`), oder weil sie gleiche Attributwerte teilen, wie `lang`. Es sollte nur verwendet werden, wenn kein anderes semantisches Element geeignet ist. `<span>` ist sehr ähnlich einem div-Element, aber div ist ein {{Glossary("Block-level_content", "Block-Level-Element")}}, während ein `<span>` ein {{Glossary("Inline-level_content", "Inline-Level-Element")}} ist.         |
| {{HTMLElement("strong")}} | Gibt an, dass seine Inhalte eine starke Bedeutung, Ernsthaftigkeit oder Dringlichkeit haben. Browser stellen typischerweise die Inhalte in Fetttyp dar.                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| {{HTMLElement("sub")}}    | Gibt Inline-Text an, der aus rein typografischen Gründen als Subskript angezeigt werden soll. Subskripte werden üblicherweise mit einem abgesenkten Baseline unter Verwendung kleinerer Schriftgröße dargestellt.                                                                                                                                                                                                                                                                                                                                                                                     |
| {{HTMLElement("sup")}}    | Gibt Inline-Text an, der aus rein typografischen Gründen als Superskript angezeigt werden soll. Superskripte werden üblicherweise mit einem angehobenen Baseline unter Verwendung kleinerer Schriftgröße dargestellt.                                                                                                                                                                                                                                                                                                                                                                                 |
| {{HTMLElement("time")}}   | Stellt eine spezifische Zeitspanne dar. Es kann das `datetime`-Attribut einschließen, um Daten in ein maschinenlesbares Format zu übersetzen und so bessere Suchmaschinenergebnisse oder benutzerdefinierte Funktionen wie Erinnerungen zu ermöglichen.                                                                                                                                                                                                                                                                                                                                               |
| {{HTMLElement("u")}}      | Stellt einen Bereich von Inline-Text dar, der so gerendert werden sollte, dass angezeigt wird, dass er eine nichttextliche Anmerkung enthält. Dies wird standardmäßig als ein einheitlicher, durchgezogener Unterstrich gerendert, kann jedoch mithilfe von CSS geändert werden.                                                                                                                                                                                                                                                                                                                      |
| {{HTMLElement("var")}}    | Stellt den Namen einer Variablen in einem mathematischen Ausdruck oder einem Programmierskontext dar. Es wird typischerweise mit einer kursiven Version der aktuellen Schriftart dargestellt, obwohl dieses Verhalten browserabhängig ist.                                                                                                                                                                                                                                                                                                                                                            |
| {{HTMLElement("wbr")}}    | Stellt eine Worttrennungsmöglichkeit dar — eine Position innerhalb des Textes, an der der Browser optional eine Zeilenumbruch einfügen kann, obwohl seine Regeln zur Zeilenumbrüchen nicht sonst an dieser Stelle einen Umbruch erzeugen würden.                                                                                                                                                                                                                                                                                                                                                      |

## Bild und Multimedia

HTML unterstützt verschiedene Multimedia-Ressourcen wie Bilder, Audio und Video.

| Element                  | Beschreibung                                                                                                                                                                                                                                                                                                                           |
| ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("area")}}  | Definiert einen Bereich innerhalb einer Imagemap, die vordefinierte klickbare Bereiche hat. Eine _Imagemap_ ermöglicht es, geometrische Bereiche auf einem Bild mit einem {{Glossary("Hyperlink", "Hyperlink")}} zu verknüpfen.                                                                                                        |
| {{HTMLElement("audio")}} | Wird verwendet, um Sound-Inhalte in Dokumente einzubetten. Es kann eine oder mehrere Audioquellen enthalten, dargestellt durch das `src`-Attribut oder das source-Element: der Browser wird den geeignetsten Inhalt wählen. Es kann auch das Ziel für gestreamte Medien sein, mit einem [`MediaStream`](/de/docs/Web/API/MediaStream). |
| {{HTMLElement("img")}}   | Betten Sie ein Bild in das Dokument ein.                                                                                                                                                                                                                                                                                               |
| {{HTMLElement("map")}}   | Wird mit {{HTMLElement("area")}}-Elementen verwendet, um eine Imagemap (einen klickbaren Link-Bereich) zu definieren.                                                                                                                                                                                                                  |
| {{HTMLElement("track")}} | Wird als Kind der Medienelemente Audio und Video verwendet. Es ermöglicht Ihnen, zeitgesteuerte Textspuren (oder zeitbasierte Daten) anzugeben, um zum Beispiel automatisch Untertitel zu verwenden. Die Spuren sind im [WebVTT-Format](/de/docs/Web/API/WebVTT_API) (`.vtt`-Dateien) - Web Video Text Tracks formatiert.              |
| {{HTMLElement("video")}} | Bettsch einen Mediaplayer ein, der die Videowiedergabe im Dokument unterstützt. Sie können `<video>` auch für Audioinhalte verwenden, aber das audio-Element bietet möglicherweise eine geeignetere Benutzererfahrung.                                                                                                                 |

## Skripting

| {{HTMLElement("noscript")}} | Definiert einen Abschnitt von HTML, der eingefügt werden soll, wenn ein Skripttyp auf der Seite nicht unterstützt wird oder wenn Skripting im Browser derzeit deaktiviert ist.                                                                                                                                                                   |
| {{HTMLElement("script")}}   | Wird verwendet, um ausführbaren Code oder Daten einzubetten; typischerweise zur Einbettung oder zum Verweisen auf JavaScript-Code verwendet. Das `<script>`-Element kann auch mit anderen Programmiersprachen verwendet werden, wie der GLSL Shader-Programmiersprache von [WebGL](/de/docs/Web/API/WebGL_API) und {{Glossary("JSON", "JSON")}}. |

## Web-Komponenten

Web Components ist eine HTML-verwandte Technologie, die es ermöglicht, im Wesentlichen benutzerdefinierte Elemente zu erstellen und zu verwenden, als wären sie reguläres HTML. Darüber hinaus können Sie benutzerdefinierte Versionen von Standard-HTML-Elementen erstellen.

| Element                     | Beschreibung                                                                                                                                                                                                                                                                         |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("slot")}}     | Teil der [Web Components](/de/docs/Web/API/Web_components)-Technologiesuite, dieses Element ist ein Platzhalter innerhalb einer Webkomponente, den Sie mit Ihrem eigenen Markup füllen können, was Ihnen ermöglicht, separate DOM-Bäume zu erstellen und sie gemeinsam darzustellen. |
| {{HTMLElement("template")}} | Ein Mechanismus zum Halten von HTML, das nicht sofort gerendert werden soll, wenn eine Seite geladen wird, aber möglicherweise während der Laufzeit später mittels JavaScript instanziiert wird.                                                                                     |
