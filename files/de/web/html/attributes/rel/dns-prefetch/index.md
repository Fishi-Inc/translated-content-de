{{HTMLSidebar}}

Das Schlüsselwort **`dns-prefetch`** für das [`rel`](/de/docs/Web/HTML/Element/link#rel)-Attribut des {{HTMLElement("link")}}-Elements ist ein Hinweis für Browser, dass der Nutzer wahrscheinlich Ressourcen vom Ursprung der Zielressource benötigt. Daher kann der Browser wahrscheinlich die Benutzererfahrung verbessern, indem er vorzeitig die DNS-Auflösung für diesen Ursprung durchführt.

Siehe [Verwendung von dns-prefetch](/de/docs/Web/Performance/dns-prefetch) für weitere Einzelheiten.
