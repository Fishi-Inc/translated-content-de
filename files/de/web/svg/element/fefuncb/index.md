Das **`<feFuncB>`** [SVG](/de/docs/Web/SVG) Filter-Primitive definiert die Übertragungsfunktion für die blaue Komponente der Eingabegrafik seines übergeordneten {{SVGElement("feComponentTransfer")}} Elements.
