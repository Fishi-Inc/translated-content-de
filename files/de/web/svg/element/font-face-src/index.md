Das **`<font-face-src>`** [SVG](/de/docs/Web/SVG)-Element entspricht dem {{cssxref("@font-face/src", "src")}} Deskriptor in CSS {{cssxref("@font-face")}} Regeln. Es dient als Container für {{SVGElement("font-face-name")}}, das auf lokal installierte Kopien dieser Schriftart verweist, und {{SVGElement("font-face-uri")}}, das remote definierte Schriftarten verwendet.
