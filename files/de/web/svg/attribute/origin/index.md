{{SVGRef}}

Das **`origin`** Attribut gibt den Ursprung der Bewegung für eine Animation an. Es hat keine Wirkung in SVG.

Sie können dieses Attribut mit den folgenden SVG-Elementen verwenden:

- {{SVGElement("animateMotion")}}
