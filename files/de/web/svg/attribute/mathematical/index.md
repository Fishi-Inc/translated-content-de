Das **`mathematisch`** Attribut gibt die Ausrichtungskoordinate für {{Glossary("glyph", "glyphs")}} an, um eine mathematische Baseline-Ausrichtung für horizontal ausgerichtete Glyphenlayouts zu erreichen. Der Wert ist ein Versatz im Koordinatensystem der Schriftart.

Sie können dieses Attribut mit den folgenden SVG-Elementen verwenden:

- {{SVGElement("font-face")}}
