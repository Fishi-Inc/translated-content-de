{{AddonSidebar}}

Die Kategorie der Warnung, die das {{WebExtAPIRef("runtime.onPerformanceWarning")}}-Ereignis ausgelöst hat.

## Typ

Werte dieses Typs sind Zeichenfolgen. Mögliche Werte sind:

- `"content_script"`: Die Leistungswarnung betrifft ein langsames Content-Script in der abhörenden Erweiterung.
