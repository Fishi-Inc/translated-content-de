{{AddonSidebar}}

Die Architektur des nativen Clients. Diese kann auf einigen Plattformen von `arch` abweichen.

## Typ

Werte dieses Typs sind Zeichenfolgen. Mögliche Werte sind: `"arm"`, `"x86-32"`, `"x86-64"`.
