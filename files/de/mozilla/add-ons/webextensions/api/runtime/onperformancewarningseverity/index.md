{{AddonSidebar}}

Der Schweregrad der Warnung, der das {{WebExtAPIRef("runtime.onPerformanceWarning")}}-Ereignis ausgelöst hat.

## Typ

Werte dieses Typs sind Zeichenfolgen. Mögliche Werte sind:

- `"low"`
- `"medium"`
- `"high"`
