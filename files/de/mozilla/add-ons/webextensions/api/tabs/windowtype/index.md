{{AddonSidebar}}

Der Typ des Fensters, das diesen Tab hostet.

## Typ

Werte dieses Typs sind Zeichenfolgen. Mögliche Werte sind:

- **"normal"**
- **"popup"**
- **"panel"**
- **"devtools"**
