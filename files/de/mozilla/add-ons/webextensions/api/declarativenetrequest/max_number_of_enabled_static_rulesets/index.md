{{AddonSidebar}}

Die maximale Anzahl statischer Regelsätze, die eine Erweiterung aktiviert haben kann. Weitere Informationen zu den Grenzen für statische Regelsätze finden Sie unter [statische Regelsatzgrenzen](/de/docs/Mozilla/Add-ons/WebExtensions/API/declarativeNetRequest#static_ruleset_limits).

Eine Erweiterung kann die Anzahl der aktivierten Regelsätze mithilfe der {{WebExtAPIRef("declarativeNetRequest.updateEnabledRulesets", "updateEnabledRulesets")}} Methode ändern.

Der Wert ist `10`.
