{{AddonSidebar}}

Gibt den Typ des zu erstellenden Browserfensters an.

## Typ

Werte dieses Typs sind `strings`. Mögliche Werte sind:

- `"normal"`
- `"popup"`
- `"panel"`
- `"detached_panel"`
