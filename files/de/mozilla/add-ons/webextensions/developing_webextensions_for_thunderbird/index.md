{{AddonSidebar}}

Sie werden die Kodierung einer Erweiterung für Thunderbird auf die gleiche Weise angehen wie bei einer Firefox-Erweiterung; indem Sie einen Texteditor oder das Tool Ihrer Wahl verwenden, um den Code zu schreiben.

## API-Unterschiede

> [!NOTE]
> Siehe [ReadTheDocs](https://webextension-api.thunderbird.net/en/latest/) für Thunderbird spezifische WebExtension-API-Dokumentation.

Da beide auf Gecko basieren, unterstützt Thunderbird viele der APIs, die auch Firefox unterstützt, mit einigen Unterschieden. Siehe [Browser-Kompatibilität für manifest.json](/de/docs/Mozilla/Add-ons/WebExtensions/Browser_compatibility_for_manifest.json) und [Browser-Unterstützung für JavaScript-APIs](/de/docs/Mozilla/Add-ons/WebExtensions/Browser_support_for_JavaScript_APIs) für Details.
